Доступны в [[bash]] через номера аргументов. Например, первый аргмуент доступен через `$1`, второй аргумент через - `$2` и так далее. Также есть специальный идентификатор - `$0`, который всегда хранит имя скрипта.

Кроме `$0` внутри скрипта можно использовать еще ряд специальных идентификаторов:
- `$#` - число аргументов
- `$@` и `$*` - строка с аргументами

``` bash
#!/bin/bash

ARG1=$1
ARG2=$2
SCRIPTNAME=$0
echo the number of arguments is $#
echo the arguments are $@
```
