Объединение условий

``` bash
команда1 && команда2 # и
команда1 || команда2 # или
```


**`Условная конструкция if..then..else`**

``` bash
if [[ условие ]]; then  
	echo "Условие выполнено" 
elif [[ условие 2 ]]; then
    echo "Условие 2 выполнено"
else  
	echo "Условие не выполнено"  
fi
```

(( условие )) - для арифметики 


**`Конструкция case`**

``` bash
case выражение in
    значение1)
    набор_команд1
    ;;
    значение2)
    набор_команд2
    ;;
    значениеN)
    набор_командN
    ;;
    значение3|значение4)
    набор_команд3и4)
    ;;
    *)
    echo "Неизвестное значение"
    ;;
esac
```


**`Цикл while`**

``` bash
while условие
do
    выполняемые_команды
done
```


**`Цикл until`**

Cинтаксически похож на `while`, только обладает обратным действием - выполниет цикл, пока не будет выполняться некоторое условие.


**`Цикл for`**

``` bash
for (( счетчик; условие; приращение_счетчика)); do
набор_команд
done
```


**`Цикл for..in`**

``` bash
for переменая in набор_значений
do
    набор_команд
done 
```
