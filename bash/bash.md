Это командная [[оболочка]] в [[Linux]]. Она содержит различный синтаксис.


Цикл for

``` bash
for (( счетчик; условие; приращение_счетчика)); do
набор_команд
done
```


**`Цикл for..in`**

``` bash
for переменая in набор_значений
do
    набор_команд
done 
```
