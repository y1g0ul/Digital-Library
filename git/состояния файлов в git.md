Каждый файл в [[git]] может находиться в одном из нескольких состояний.
``` bash
Untracked # Файл существует в рабочей директории, но Git о нём ничего не знает.
     │
git add
     ▼
Staged # Файл находится в индексе (staging area) и будет включён в следующий коммит.
     │
git commit
     ▼
Tracked (Unmodified) # Git отслеживает файл. Пока его не изменяли, он находится в состоянии Unmodified.
     │
изменение файла
     ▼
Modified # Файл отслеживается, но был изменён после последнего коммита.
     │
git add
     ▼
Staged
```
