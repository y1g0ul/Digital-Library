Команда для просмотра объектов [[git]] (коммиты, деревья, файлы) по их хешу.
``` bash
git cat-file <опция> <объект>
```

|Опция|Назначение|
|---|---|
|`-p`|Показать содержимое объекта|
|`-t`|Показать тип объекта|
|`-s`|Показать размер объекта|
