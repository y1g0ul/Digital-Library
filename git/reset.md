Команда для изменения состояния [[git]]: перемещения указателя [[git/HEAD|HEAD]], изменения Staging Area и отмены [[commit]]ов. Отменяет изменения, перенося ссылку на [[branch]] назад, на старый [[commit]], как будто некоторых [[commit]]ов вовсе и не было. Используется для локальных [[branch]].
``` bash
git reset <опция> <commit>
```

| Опция     | Назначение                                                         |
| --------- | ------------------------------------------------------------------ |
| `--soft`  | Удаляет коммиты, но оставляет изменения в Staging Area             |
| `--mixed` | Удаляет коммиты и убирает изменения из Staging Area (по умолчанию) |
| `--hard`  | Удаляет коммиты и изменения из рабочей директории                  |
| `--keep`  | Удаляет коммиты, сохраняя локальные изменения                      |
![[git reset]]
``` bash
git reset --soft HEAD~1 # Отменить последний коммит, оставить изменения подготовленными
git reset --mixed HEAD~1 # Отменить последний коммит, оставить изменения в файлах
git reset --hard HEAD~1 # Полностью удалить последний коммит и изменения
git reset HEAD file.txt # Убрать файл из Staging Area
```
