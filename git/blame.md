Команда для просмотра информации о том, кто и когда изменил каждую строку файла в [[git]].
``` bash
git blame <опция> <файл>
```

| Опция                 | Назначение                                        |
| --------------------- | ------------------------------------------------- |
| `-L <начало>,<конец>` | Показать указанный диапазон строк                 |
| `-e`                  | Показать email автора                             |
| `-w`                  | Игнорировать изменения только пробельных символов |
| `--date=<format>`     | Изменить формат отображения даты                  |
``` bash
git blame main.py # Показать авторов всех строк файла
git blame -L 10,20 main.py # Показать строки с 10 по 20
git blame -w main.py # Игнорировать изменения пробелов
git blame -e main.py # Показать email авторов
```
