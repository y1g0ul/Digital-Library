Хранит снимок всех файлов директории  [[git]]
``` bash
git commit -m "Сообщение коммита"
```
- Cохраняет только изменения, находящиеся в `Staging Area`.
- Каждый коммит имеет уникальный [[хеш]] (SHA-1/SHA-256 в новых репозиториях).
- Сообщение коммита должно кратко описывать внесённые изменения.
