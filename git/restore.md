Команда для отмены изменений в файлах и восстановления их состояния из коммита или `Staging Area`.
``` bash
git restore <файл>
```

|Опция|Назначение|
|---|---|
|`--staged`|Убрать файл из Staging Area|
|`--source <commit>`|Восстановить файл из указанного коммита|
|`.`|Применить ко всем файлам в текущей директории|
