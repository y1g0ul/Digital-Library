Обьединяет ветки в [[git]]. Создает особый вид [[commit]]а, который имеет сразу вдух родителей. Обычно означает, что мы хотим объединить изменения из одного [[commit]]а с другим [[commit]]ом и всеми их родительскими [[commit]]ами.

``` bash
git merge <опция> <ветка>
```

![[git merge]]

| Опция        | Назначение                                                 |
| ------------ | ---------------------------------------------------------- |
| `--no-ff`    | Всегда создавать отдельный merge-коммит                    |
| `--ff-only`  | Выполнить merge только при fast-forward                    |
| `--squash`   | Объединить изменения без создания merge-коммита [[squash]] |
| `--abort`    | Отменить незавершённый merge                               |
| `--continue` | Продолжить merge после разрешения конфликтов               |
``` bash
git merge feature # Объединить ветку feature с текущей
git merge --no-ff feature # Создать отдельный merge-коммит
git merge --squash feature # Добавить изменения как один коммит
git merge --abort # Отменить конфликтный merge
git merge --continue # Продолжить после решения конфликтов
```
