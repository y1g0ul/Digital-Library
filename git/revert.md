Команда для отмены изменений [[commit]] через создание нового [[commit]]. Делает новый [[commit]], который содержит изменения полностью противоположные тем, что сделаны в последнем [[commit]]е. Используется для удаленных [[branch]] над которыми работают еще пользователи.
``` bash
git revert <commit>
```

|Опция|Назначение|
|---|---|
|`--no-edit`|Использовать стандартное сообщение коммита|
|`-n` / `--no-commit`|Отменить изменения без создания коммита|
