Команда для настройки параметров [[git]] (имя пользователя, email, редактор, алиасы и другие настройки).
``` bash
git config <опция> <параметр> <значение>
```

|Опция|Назначение|
|---|---|
|`--global`|Настройки для всех репозиториев пользователя|
|`--local`|Настройки только текущего репозитория|
|`--system`|Настройки для всей системы|
|`--list`|Показать все настройки|
|`--get`|Получить значение параметра|
|`--add`|Добавить новое значение параметра|
|`--unset`|Удалить значение параметра|
|`--unset-all`|Удалить все значения параметра|
|`--edit`|Открыть файл конфигурации в редакторе|

| Параметр      | Назначение                |
| ------------- | ------------------------- |
| `user.name`   | Имя автора коммитов       |
| `user.email`  | Email автора коммитов     |
| `core.editor` | Редактор Git по умолчанию |
| `alias.<имя>` | Создание команды-алиаса   |
