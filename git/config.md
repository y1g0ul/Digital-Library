Команда для настройки параметров [[git]] (имя пользователя, email, редактор, алиасы и другие настройки).
``` bash
git config <уровень> <параметр> <значение>
```

|Уровень|Назначение|
|---|---|
|`--local`|Настройки только текущего репозитория (по умолчанию)|
|`--global`|Настройки для всех репозиториев пользователя|
|`--system`|Настройки для всей системы|

|Параметр|Назначение|
|---|---|
|`user.name`|Имя автора коммитов|
|`user.email`|Email автора коммитов|
|`core.editor`|Редактор Git по умолчанию|
|`alias.<имя>`|Создание команды-алиаса|
