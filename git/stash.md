Команда для временного сохранения незакоммиченных изменений, чтобы переключиться на другую [[branch]] или выполнить другие действия.
``` bash
git stash <опция>
```

| Опция   | Назначение                              |
| ------- | --------------------------------------- |
| `push`  | Создать новый stash                     |
| `list`  | Показать список stash                   |
| `pop`   | Применить последний stash и удалить его |
| `apply` | Применить stash без удаления            |
| `drop`  | Удалить stash                           |
| `clear` | Удалить все stash                       |
| `-u`    | Сохранить также untracked-файлы         |
| `-m`    | Добавить сообщение к stash              |
``` bash
git stash # Сохранить текущие изменения
git stash push -m "WIP feature" # Сохранить изменения с описанием
git stash list # Показать сохранённые изменения
git stash pop # Вернуть последний stash и удалить его
git stash apply # Вернуть stash без удаления
git stash drop stash@{0} # Удалить конкретный stash
git stash clear # Удалить все stash
git stash -u # Сохранить вместе с новыми файлами
```
