Команда показывающая как далеко текущее состояние от ближайшего тега. (`ref` — это что-либо, что указывает на конкретный коммит. Если не указать `ref`, то git будет считать, что указано текущее положение [[git/HEAD|HEAD]].)
``` bash
git describe <опция> <ref>
```
Вывод команды выглядит примерно так:
`<tag>-<numCommits>-g<hash>`

Где `tag` – это ближайший тег в истории изменений, `numCommits` – это на сколько далеко мы от этого тега, а `hash` – это хеш [[commit]]а, который описывается.

|Опция|Назначение|
|---|---|
|`--tags`|Использовать все теги, включая lightweight|
|`--all`|Искать среди всех ссылок (ветки, теги)|
|`--long`|Всегда показывать количество коммитов и хеш|
|`--abbrev=<n>`|Задать длину сокращённого хеша|
``` bash
git describe # Показать описание текущего коммита
git describe --tags # Искать среди всех тегов
git describe --long # Показать полный формат с количеством коммитов
```
