Команда для просмотра истории [[commit]]ов в [[git]].
``` bash
git log <опция>
```

|Опция|Назначение|
|---|---|
|`--oneline`|Краткий вывод коммитов в одну строку|
|`--graph`|Показать историю в виде графа|
|`--all`|Показать историю всех веток|
|`-n <число>`|Показать последние N коммитов|
|`--author=<имя>`|Фильтр по автору|
|`--since=<дата>`|Коммиты после указанной даты|
|`--until=<дата>`|Коммиты до указанной даты|
|`--stat`|Показать статистику изменений|
