Команда для просмотра истории [[commit]]ов в [[git]].
``` bash
git log <опция>
```

|Опция|Назначение|
|---|---|
|`--oneline`|Краткий вывод коммитов|
|`--graph`|Показать историю в виде графа|
|`--all`|Показать историю всех веток|
|`-n <число>`|Показать последние N коммитов|
|`--author=<имя>`|Фильтр по автору|
|`--since=<дата>`|Коммиты после указанной даты|
|`--until=<дата>`|Коммиты до указанной даты|
|`--stat`|Показать статистику изменений|
|`-p`|Показать изменения (diff) каждого коммита|
|`--grep=<текст>`|Найти коммиты по сообщению|
``` bash
git log # Полная история коммитов
git log --oneline # Краткая история
git log --graph --oneline --all # История всех веток в виде графа
git log -5 # Последние 5 коммитов
git log --author="Nikita" # Коммиты указанного автора
git log --since="2026-07-01" # Коммиты после указанной даты
git log --stat # История со статистикой изменений
git log -p # История с diff каждого коммита
git log -- README.md # История изменений файла README.md
```

Для повседневной работы чаще всего используются:
- `git log --oneline`
- `git log --graph --oneline --all`
- `git log -p`
- `git log -- <файл>`
