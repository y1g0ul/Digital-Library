Команда в [[git]] которая по сути заменяет собой 2 другие команды [[fetch]] и [[слияние merge]]. Она вытягивет изменения из удаленного репозитория и обьядиняет с нашим. 

![[git pull]]

Флаг `--rebase` позволит команде выполняться не по сценарию [[слияние merge]], а по [[слияние rebase]].

Так же можно добавтить аргументы `<источник>:<получатель>`.
