Команда для переключения между [[branch]] и восстановления файлов в [[git]].
``` bash
git checkout <ветка/файл>
```

| Опция | Назначение                                                                                                                                                                                   |
| ----- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `-b`  | Создать новую ветку и переключиться на неё. Так же можно создать [[branch]] которая будет отслеживать [[branch]] из удаленного репозитория - `git checkout -b totallyNotMain origin/main` .  |
| `--`  | Указать, что дальше идёт имя файла                                                                                                                                                           |
