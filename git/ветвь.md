В [[git]] это просто ссылки на определенный [[коммит]].

`git branch [name]` - создать ветку
- `-f` - передвинуть [[ветвь]] на другой [[коммит]] (`git branch -f main HEAD~2` - Передвинуть main на 2 [[коммит]]а назад от [[git/head|head]])
- `-u` - указать [[ветвь]] и отслеживать удаленную [[ветвь]] (`git branch -u o/main foo`, а если находиться на ветке `foo`, то её можно не указывать - `git branch -u o/main`)

`git checkout [name]` - переключиться на новую ветку или [[коммит]]
- ` -b [yourbranchname]` - создать новую ветку и сразу на нее переключиться. Так же можно создать [[ветвь]] которая будет отслеживать [[ветвь]] из удаленного репозитория - `git checkout -b totallyNotMain origin/main` .
