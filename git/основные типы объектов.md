[[git]] ранит данные в виде объектов внутри `.git/objects`.

| Объект     | Описание                                                              |
| ---------- | --------------------------------------------------------------------- |
| **Blob**   | Содержимое файла                                                      |
| **Tree**   | Список файлов и директорий (структура проекта)                        |
| [[commit]] | Снимок состояния проекта + информация об авторе, сообщении и родителе |
| [[tag]]    | Именная ссылка на конкретный объект (обычно commit)                   |
