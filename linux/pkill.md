Утилита в [[Linux]] для завершения процессов по имени или другим критериям.  
Работает как [[pgrep]], но отправляет сигнал процессу. По умолчанию отправляет `SIGTERM`.

`pkill [опции] шаблон`

|Опция|Описание|
|---|---|
|`-9`|Отправить `SIGKILL`|
|`-u user`|Завершить процессы пользователя|
|`-x`|Точное совпадение имени|
|`-f`|Искать по всей командной строке|
|`-n`|Завершить самый новый процесс|
|`-o`|Завершить самый старый процесс|
|`-l`|Показать, какие процессы будут завершены|
``` bash
pkill nginx               # завершить все процессы nginx
pkill -9 nginx            # принудительное завершение
pkill -u user             # завершить процессы пользователя
pkill -f python           # завершить по командной строке
pkill -x sshd             # точное совпадение имени
```
