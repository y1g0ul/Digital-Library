Команда в [[Linux]] которая обьединяет файлы прострочно
``` bash
paste [опции] файл1 файл2 ...
```

| Опция          | Описание                                      |
| -------------- | --------------------------------------------- |
| `-d <символы>` | Задать разделитель(и) вместо TAB              |
| `-s`           | Последовательно склеивать строки одного файла |
| `--serial`     | То же самое, что `-s`                         |
| `--help`       | Показать справку                              |
| `--version`    | Версия утилиты                                |
``` bash
paste users.txt emails.txt        # объединить файлы построчно через TAB
paste -d ':' users.txt emails.txt # объединить, используя ':' как разделитель
paste -d ',;' f1 f2               # разные разделители по кругу
paste -s users.txt                # склеить все строки файла в одну
paste -s -d ' ' users.txt         # склеить строки через пробел
```
