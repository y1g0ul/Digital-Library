Специальные обозначения в [[Linux]] для навигации по файловой системе.  
Используются при работе с `cd` и при указании путей.

| Обозначение    | Описание                               |
| -------------- | -------------------------------------- |
| `~` или `~/`   | Домашний каталог текущего пользователя |
| `.` или `./`   | Текущий каталог                        |
| `..` или `../` | Родительский каталог                   |
| `~user`        | Домашний каталог пользователя `user`   |
| `~-`           | Предыдущий посещённый каталог          |
| `~+`           | Текущий каталог (значение `$PWD`)      |

``` bash
cd ~                 # перейти в домашний каталог
cd ~/projects        # путь относительно домашнего каталога
cd .                 # остаться в текущем каталоге
cd ./folder          # перейти в папку внутри текущего каталога
cd ..                # перейти в родительский каталог
cd ../..             # подняться на два уровня выше
cd ~root             # перейти в домашний каталог root
cd -                 # вернуться в предыдущий каталог
echo ~+              # вывести текущий путь
echo ~-              # вывести предыдущий путь
```
