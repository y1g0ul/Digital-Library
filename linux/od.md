Octal dump это команда в [[Linux]] предназначена для чтения данных из файла или стандартного ввода и вывода их содержимого в различных форматах: восьмеричном (по умолчанию), шестнадцатеричном, десятичном или ASCII. Показывает **сырые байты файла** в читаемом виде (числа или символы)
``` bash
od [опции] файл
```

| Ключ  | Что делает                                             |
| ----- | ------------------------------------------------------ |
| `-c`  | вывод в символьном виде                                |
| `-a`  | символьные имена (control-символы)                     |
| `-t`  | задать формат вывода (x — hex, o — octal, d — decimal) |
| `-x`  | вывод в hex                                            |
| `-o`  | вывод в octal                                          |
| `-d`  | вывод в decimal                                        |
| `-An` | убрать адреса в выводе                                 |
| `-N`  | прочитать только N байт                                |
| `-j`  | пропустить N байт                                      |
``` bash
od file.bin                  # стандартный octal вывод

od -c file.txt               # читаемый символьный вывод

od -x file.bin               # hex формат

od -An -t x1 file.bin        # hex без адресов (удобно для анализа)

od -N 16 file.bin            # первые 16 байт

od -j 32 -N 16 file.bin      # пропустить 32 байта и читать 16
```
