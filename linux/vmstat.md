Команда в [[Linux]] для просмотра статистики виртуальной памяти, [[процесс]]ов, процессора, ввода-вывода и использования памяти. 
``` bash
vmstat [опции] [интервал] [количество]
```

|Ключ|Что делает|
|---|---|
|`-a`|показать активную и неактивную память|
|`-d`|статистика дисков|
|`-D`|сводная статистика дисков|
|`-f`|количество созданных процессов с момента загрузки|
|`-m`|статистика slab-кэша|
|`-s`|сводная статистика памяти|
|`-w`|широкий формат вывода|
|`-t`|добавить временную метку|
|`-S`|выбрать единицы измерения (`k`, `K`, `m`, `M`)|
|`-n`|не повторять заголовок при обновлении|
``` bash
vmstat
vmstat 2
vmstat 2 5
vmstat -a
vmstat -ss
vmstat -d
```
