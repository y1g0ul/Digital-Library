Команда в [[Linux]] для изменения параметров пользователя
``` bash
usermod [опции] пользователь
```

| Ключ  | Что делает                                                      |
| ----- | --------------------------------------------------------------- |
| `-aG` | добавить в группу                                               |
| `-G`  | перезаписать группы                                             |
| `-l`  | изменить имя пользователя (логин)                               |
| `-d`  | изменить домашний каталог                                       |
| `-m`  | переместить содержимое домашнего каталога (используется с `-d`) |
| `-s`  | изменить shell                                                  |
| `-u`  | изменить UID пользователя                                       |
| `-g`  | изменить основную группу                                        |
| `-L`  | заблокировать пользователя                                      |
| `-U`  | разблокировать пользователя                                     |
| `-e`  | задать дату истечения аккаунта                                  |
``` bash
usermod -aG sudo y1g0ul        # добавить пользователя в группу sudo
usermod -G docker y1g0ul       # задать только группу docker 

usermod -l newname y1g0ul      # изменить логин
usermod -d /home/newname -m newname  # сменить домашний каталог и переместить файлы

usermod -s /bin/zsh y1g0ul     # изменить shell

usermod -L y1g0ul              # заблокировать пользователя
usermod -U y1g0ul              # разблокировать пользователя

usermod -u 2000 y1g0ul         # изменить UID
usermod -g users y1g0ul        # изменить основную группу
```
