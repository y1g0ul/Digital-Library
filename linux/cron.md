Crontab (от "cron table") — это системный файл (или команда) в [[Linux]], который позволяет запускать программы или скрипты по расписанию.  Он может работать не чаще чем раз в минуту.

Конфигурационный файл для общего крона находится в `/etc/crontab`

`* * * * * пользователь команда`

Индивидуальные в каталоге `/var/spool/cron/crontabs` 
`* * * * * команда_или_скрипт`


Где звёздочки (`*`) означают:  

| Поле  | Допустимые значения         | Описание                      |
| ----- | --------------------------- | ----------------------------- |
| **1** | `0-59`                      | Минуты (когда выполнять)      |
| **2** | `0-23`                      | Часы (когда выполнять)        |
| **3** | `1-31`                      | День месяца (когда выполнять) |
| **4** | `1-12`                      | Месяц (когда выполнять)       |
| **5** | `0-7` (0 и 7 = воскресенье) | День недели (когда выполнять) |
``` bash
* * * * * команда
│ │ │ │ │
│ │ │ │ └── день недели (0-7)
│ │ │ └──── месяц (1-12)
│ │ └────── день месяца (1-31)
│ └──────── час (0-23)
└────────── минута (0-59)
```

Специальные символы

|Символ|Пример|Объяснение|
|---|---|---|
|`*`|`* * * * *`|Любое значение (каждую минуту)|
|`,`|`0,15,30,45 * * * *`|В 0, 15, 30 и 45 минут каждого часа|
|`-`|`0 9-18 * * *`|Каждый час с 9:00 до 18:00|
|`*/n`|`*/10 * * * *`|Каждые 10 минут|
|`@reboot`|`@reboot /home/user/start.sh`|Выполнить скрипт при загрузке системы|
|`@hourly`|`@hourly /home/user/backup.sh`|Выполнять скрипт каждый час|
|`@daily`|`@daily /home/user/backup.sh`|Выполнять скрипт каждый день|
|`@weekly`|`@weekly /home/user/report.sh`|Выполнять скрипт каждую неделю|
|`@monthly`|`@monthly /home/user/clean.sh`|Выполнять скрипт каждый месяц|
|`@yearly`|`@yearly /home/user/archive.sh`|Выполнять скрипт каждый год|
Примеры:
- `0 * * * * /backup.sh` → Каждый час, в :00 (например, 12:00, 13:00...)  
- `30 3 * * * /cleanup.sh` → Каждый день в 3:30 утра
- `0 0 * * 0 /reboot.sh` → Каждое воскресенье в 00:00
- `*/5 * * * * /ping_server.sh` → Каждые 5 минут

Файлы управления доступом:  
- **`/etc/cron.allow`**  
- **`/etc/cron.deny`**

Они определяют, кому разрешено использовать команду.  Если существует `/etc/cron.allow`. Тогда использовать `crontab` могут только пользователи, перечисленные в этом файле. Если есть `/etc/cron.deny`. Тогда использовать [[cron]] могут все пользователи кроме перечисленных. Если существуют оба файла. Приоритет имеет `cron.allow`.
