Crontab (от "cron table") — это системный файл (или команда) в [[Linux]], который позволяет запускать программы или скрипты по расписанию.  Он может работать не чаще чем раз в минуту.

Конфигурационный файл для общего крона находится в `/etc/crontab`
`* * * * * пользователь команда`

Индивидуальные в каталоге `/var/spool/crontabs` 
`* * * * * команда_или_скрипт`


Где звёздочки (`*`) означают:  

| Поле  | Допустимые значения         | Описание                      |
| ----- | --------------------------- | ----------------------------- |
| **1** | `0-59`                      | Минуты (когда выполнять)      |
| **2** | `0-23`                      | Часы (когда выполнять)        |
| **3** | `1-31`                      | День месяца (когда выполнять) |
| **4** | `1-12`                      | Месяц (когда выполнять)       |
| **5** | `0-7` (0 и 7 = воскресенье) | День недели (когда выполнять) |

Специальные символы

| Символ | Пример               | Объяснение                          |
| ------ | -------------------- | ----------------------------------- |
| `*`    | `* * * * *`          | Любое значение (каждую минуту)      |
| `,`    | `0,15,30,45 * * * *` | В 0, 15, 30 и 45 минут каждого часа |
| `-`    | `0 9-18 * * *`       | Каждый час с 9:00 до 18:00          |
| `*/n`  | `*/10 * * * *`       | Каждые 10 минут                     |

Примеры:
- `0 * * * * /backup.sh` → Каждый час, в :00 (например, 12:00, 13:00...)  
- `30 3 * * * /cleanup.sh` → Каждый день в 3:30 утра
- `0 0 * * 0 /reboot.sh` → Каждое воскресенье в 00:00
- `*/5 * * * * /ping_server.sh` → Каждые 5 минут

Основные команды crontab: 
-  `crontab -e` Редактировать текущий crontab (откроется в текстовом редакторе) 
- `crontab -l`  Показать текущие задачи 
- `crontab -r` Удалить все задачи (осторожно!) 
- `crontab -u user` Управление crontab другого пользователя (требует прав root)
