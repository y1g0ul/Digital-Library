Система управления пакетами в [[Linux]].

**`apt`** — удобный интерфейс для человека

| Команда                 | Что делает                                    |
| ----------------------- | --------------------------------------------- |
| `apt update`            | Обновить список пакетов из репозиториев       |
| `apt upgrade`           | Обновить установленные пакеты (без удаления)  |
| `apt full-upgrade`      | Обновить пакеты с возможным удалением/заменой |
| `apt install pkg`       | Установить пакет                              |
| `apt remove pkg`        | Удалить пакет (конфиги оставить)              |
| `apt purge pkg`         | Удалить пакет **и конфиги**                   |
| `apt autoremove`        | Удалить ненужные зависимости                  |
| `apt search name`       | Поиск пакета                                  |
| `apt show pkg`          | Информация о пакете                           |
| `apt list --installed`  | Список установленных пакетов                  |
| `apt list --upgradable` | Что можно обновить                            |

**`apt-get`** — стабильный интерфейс для скриптов и серверов

|Команда|Что делает|
|---|---|
|`apt-get update`|Обновить индекс пакетов|
|`apt-get upgrade`|Обновить без удаления пакетов|
|`apt-get dist-upgrade`|Обновление с заменой / удалением пакетов|
|`apt-get install pkg`|Установить пакет|
|`apt-get remove pkg`|Удалить пакет|
|`apt-get purge pkg`|Удалить пакет и конфиги|
|`apt-get autoremove`|Удалить лишние зависимости|
|`apt-get -f install`|Починить зависимости|
|`apt-get clean`|Очистить кеш `.deb`|
|`apt-get autoclean`|Очистить устаревшие пакеты|
**`apt-cache`** — поиск и анализ пакетов (без установки)

|Команда|Что делает|
|---|---|
|`apt-cache search name`|Поиск пакетов по имени/описанию|
|`apt-cache show pkg`|Подробная информация о пакете|
|`apt-cache depends pkg`|Зависимости пакета|
|`apt-cache rdepends pkg`|Какие пакеты зависят от него|
|`apt-cache policy pkg`|Версии пакета и источник|

**`aptitude`** — интерактивное управление пакетами. Альтернатива `apt/apt-get` с **псевдографическим интерфейсом**

Репозитории APT
- `/etc/apt/sources.list.d/ubuntu.sources` - основной файл со списком источников пакетов
- `/etc/apt/sources.list` - основной файл со списком источников пакетов в старых системах
- `/etc/apt/sources.list.d/*.list` - дополнительные репозитории
