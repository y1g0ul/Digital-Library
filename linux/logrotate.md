Команда в [[Linux]] для автоматической ротации, сжатия и удаления логов.  Используется в ежедневном планировщие [[cron]]. 
``` bash
logrotate [опции] конфигурация
```

| Ключ      | Что делает                            |
| --------- | ------------------------------------- |
| `-d`      | отладочный режим (ничего не изменяет) |
| `-f`      | принудительно выполнить ротацию       |
| `-v`      | подробный вывод                       |
| `-s`      | указать файл состояния                |
| `--state` | то же самое, что `-s`                 |
Основной конфигурационный файл `/etc/logrotate.conf`. Дополнительные конфигурации `/etc/logrotate.d/`.
