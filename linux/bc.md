Команда в [[Linux]] для вычислений и работы с числами с плавающей точкой

``` bash
bc [опции]
```

| Ключ    | Что делает                           |
| ------- | ------------------------------------ |
| `-l`    | подключить математическую библиотеку |
| `-q`    | не показывать приветствие            |
| `scale` | количество знаков после запятой      |
| `ibase` | основание входных чисел              |
| `obase` | основание выходных чисел             |
``` bash

echo "2 + 2" | bc                     # сложение

echo "10 / 3" | bc                    # целочисленное деление

echo "scale=2; 10 / 3" | bc           # деление с дробной частью

echo "2 ^ 10" | bc                    # степень

echo "sqrt(25)" | bc -l               # квадратный корень

echo "s(1)" | bc -l                   # синус

echo "ibase=2; 1010" | bc             # бинарное число в decimal

echo "obase=16; 255" | bc             # decimal в hex
```
