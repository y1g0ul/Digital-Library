Команда в [[Linux]] которая определяет тип содержимого файла по характерным сигнатурам.
``` bash
file [опции] файл
```

| Ключ | Что делает                             |
| ---- | -------------------------------------- |
| `-b` | выводить только тип файла без имени    |
| `-i` | показать MIME-тип                      |
| `-k` | показать все найденные совпадения      |
| `-L` | следовать по символическим ссылкам     |
| `-s` | анализировать специальные файлы        |
| `-z` | анализировать содержимое сжатых файлов |
| `-f` | прочитать список файлов из файла       |
``` bash
file test.txt                 # определить тип файла

file image.jpg                # определить тип изображения

file archive.tar.gz           # определить тип архива

file -b test.txt              # только тип файла

file -i test.txt              # MIME-тип

file -L symlink               # анализировать цель ссылки

file -z backup.gz             # анализировать содержимое архива

file *                        # определить тип всех файлов в каталоге
```
