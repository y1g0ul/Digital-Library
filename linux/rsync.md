Команда в [[Linux]] для синхронизации и копирования файлов и каталогов локально или по сети. В отличие от [[cp]], rsync копирует только изменённые части файлов, что значительно экономит время и сетевой трафик.

```
Источник
    ↓
Сравнение файлов
    ↓
Передача только изменений
    ↓
Назначение
```

``` bash
rsync [опции] источник назначение
```

``` bash
rsync file.txt backup/

rsync -av source/ backup/

rsync -avh source/ backup/

rsync -avP source/ backup/

rsync -av --delete source/ backup/

rsync -av --exclude "*.log" source/ backup/

rsync -av -e ssh source/ user@server:/backup/

rsync -av user@server:/backup/ local/

rsync -avn source/ backup/
# пробный запуск
```

  Особенность работы с каталогами:
  - Без завершающего /:
``` bash
     rsync -av dir backup/
     # Результат:
     backup/
      └── dir/
```
- С завершающим /:
``` bash 
rsync -av dir/ backup/
# Результат: 
backup/
├── file1
├── file2
└── ...
```
