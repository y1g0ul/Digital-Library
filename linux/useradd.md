Команда в [[Linux]] для создания нового пользователя
``` bash
useradd [опции] пользователь
```

| Ключ | Что делает                                           |
| ---- | ---------------------------------------------------- |
| `-m` | создать домашний каталог                             |
| `-d` | указать домашний каталог                             |
| `-s` | задать shell                                         |
| `-u` | задать UID                                           |
| `-g` | задать основную группу                               |
| `-G` | задать дополнительные группы                         |
| `-c` | комментарий (обычно имя пользователя)                |
| `-e` | дата истечения аккаунта                              |
| `-f` | количество дней до блокировки после истечения пароля |
| `-p` | задать пароль (в зашифрованном виде!)                |
``` bash
useradd y1g0ul                # создать пользователя без домашнего каталога

useradd -m y1g0ul             # создать пользователя с домашним каталогом

useradd -m -s /bin/bash y1g0ul    # создать с bash shell

useradd -m -G sudo,docker y1g0ul  # добавить в группы

useradd -u 2000 y1g0ul        # задать UID

useradd -m -d /home/custom y1g0ul # задать домашний каталог

useradd -c "Y1g0ul User" y1g0ul   # комментарий

useradd -e 2026-12-31 y1g0ul      # дата истечения аккаунта
```
