Команда в [[Linux]] для подключения (монтирования) файловой системы к дереву каталогов.
``` shell
mount [опции] устройство каталог
```
Если мы хотим что бы разделы примонтировались постоянно нам нужно добавить их в [[fstab]].

|Опция|Описание|
|---|---|
|`-t`|указать тип файловой системы|
|`-o`|дополнительные параметры монтирования|
|`-r`|монтировать только для чтения|
|`-w`|монтировать для чтения и записи|
|`-a`|смонтировать всё из `/etc/fstab`|
|`--bind`|смонтировать каталог в другой каталог|
``` bash
mount /dev/sdb1 /mnt
# подключить раздел к каталогу /mnt

mount -t ext4 /dev/sdb1 /mnt
# указать тип файловой системы

mount -o ro /dev/sdb1 /mnt
# смонтировать только для чтения

mount -o rw /dev/sdb1 /mnt
# смонтировать с правом записи

mount -t iso9660 image.iso /mnt
# смонтировать ISO образ

mount --bind /home/user/data /mnt
# смонтировать каталог в другое место

mount -a
# смонтировать все файловые системы из /etc/fstab
```
Что бы отключить есть команда
```
umount <точка_монтирования>
```
