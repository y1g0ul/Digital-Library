Команда в [[Linux]] для подключения (монтирования) файловой системы к дереву каталогов.
``` shell
mount <устройство> <точка_монтирования>
```
Если мы хотим что бы разделы примонтировались постоянно нам нужно добавить их в [[fstab]].

Что бы отключить есть команда
```
umount <точка_монтирования>
```
