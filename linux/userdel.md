Команда в [[Linux]] для удаления пользователя

``` bash
userdel [опции] пользователь
```

|Ключ|Что делает|
|---|---|
|`-r`|удалить домашний каталог и почтовый ящик пользователя|
|`-f`|принудительное удаление, даже если пользователь вошёл в систему|
|`-Z`|удалить SELinux-сопоставление пользователя (если используется SELinux)|
``` bash
userdel y1g0ul                 # удалить пользователя (удаляет только запись о пользователе из системы. Домашний каталог и файлы пользователя останутся на диске)
userdel -r y1g0ul              # удалить пользователя и домашний каталог
userdel -f y1g0ul              # принудительное удаление
userdel -r -f y1g0ul           # удалить пользователя со всеми файлами
```
