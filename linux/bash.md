Это командная [[оболочка]] в [[Linux]]. Она содержит различный синтаксис.

**`Аргументы`**

Доступны через номера аргументов. Например, первый аргмуент доступен через `$1`, второй аргумент через - `$2` и так далее. Также есть специальный идентификатор - $0, который всегда хранит имя скрипта.

Кроме `$0` внутри скрипта можно использовать еще ряд специальных идентификаторов:
- `$#` - число аргументов
- `$@` и `$*` - строка с аргументами

``` bash
#!/bin/bash

ARG1=$1
ARG2=$2
SCRIPTNAME=$0
echo the number of arguments is $#
echo the arguments are $@
```


**`Подстановка команд`**

``` bash
docker stop $(docker ps -q)
```

Перед выполнением некоторых действий может потребоваться проверить, действительно ли переменной присвоено некоторое значение. Для этого Bash предлагает операторы подстановки:

- `${parameter:-value}`: показывает значение value, если параметр не определен.
``` bash
echo ${NIL:-Variable NIL undefined}
Variable NIL undefined
```

- `${parameter=value}`: присваивает значение value параметру, если параметр вообще не существует. Этот оператор ничего не делает, если параметр существует, но не имеет значения.
``` bash
echo ${NIL=default}
default
```

- `${parameter:=value}`: присваивает значение value параметру, если этот параметр в данный момент не имеет значения или вообще не существует.
``` bash
# Выше для переменной NIL было установлено значение "default". То есть переменная NIL существует и имеет значение.
echo ${NIL:=null}
default

NIL=
echo ${NIL:=null}
null

```

- `${parameter:?value}`: показывает значение value, если параметр не существует или пуст. Использование этой конструкции приведет к немедленному прерыванию сценария оболочки.
``` bash
echo ${VOID:?Variable VOID is undefined}
bash: VOID: Variable VOID is undefined
```

- `${parameter:+value}`: показывает значение value, если параметр имеет значение. Если у параметра нет значения, ничего не происходит.
``` bash
echo ${NIL:+Variable NIL is defined}
Variable NIL is defined
echo ${VOID:+Variable VOID is defined}

```


**`Операторы сопоставления с шаблоном`**

Позволяют найти в переменной некоторый шаблон и, если этот этот шаблон найден, изменить переменную.
- Оператор `##` используется для поиска самого длинного соответствия шаблону с начала строки
``` bash
#!/bin/bash
# скрипт, который получает имя файла
filename=${1##*/}
echo "The filename is $filename"
exit 0
```

``` bash
bash hello /home/eugene/Documents/hello.txt
The filename is hello.txt
```

- Оператор `#` используется для поиска самого короткого соответствия шаблону с начала строки
``` bash
#!/bin/bash
# скрипт, который получает имя файла
path=${1#*/}
echo "path: $path"
exit 0
```

``` bash
bash hello /home/eugene/Documents/hello.txt
path: home/eugene/Documents/hello.txt
```

- Оператор `%%` используется для поиска самого длинного соответствия шаблону с конца строки
- Оператор `%` используется для поиска самого короткого соответствия шаблону с конца строки
