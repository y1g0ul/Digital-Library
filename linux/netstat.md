Команда в [[Linux]] для просмотра сетевых соединений и статистики

``` bash
netstat [опции]
```

|Ключ|Что делает|
|---|---|
|`-t`|TCP соединения|
|`-u`|UDP соединения|
|`-l`|только слушающие порты|
|`-n`|не преобразовывать адреса и порты в имена|
|`-p`|показать PID и процесс|
|`-a`|все соединения и порты|
|`-r`|таблица маршрутизации|
|`-i`|статистика интерфейсов|
|`-s`|сетовая статистика|
|`-c`|обновлять вывод постоянно|

``` bash
netstat                        # все соединения
netstat -tuln                  # слушающие TCP/UDP порты
netstat -tulpn                 # порты + процессы
netstat -an                    # все соединения без DNS
netstat -rn                    # таблица маршрутизации
netstat -i                     # сетевые интерфейсы
netstat -s                     # статистика протоколов
netstat -tulnp | grep 80       # кто слушает 80 порт
```
