Data duplicator/definition низкоуровневая утилита в [[Linux]] для **копирования** и **преобразования** данных.  
Используется для:
- создания **образов** дисков
- клонирования разделов
- работы с потоками байтов
- **тестирования скорости** устройств

``` bash
dd if=<источник> of=<приёмник> [опции]
```

|Опция|Что делает|
|---|---|
|`if=`|файл-источник|
|`of=`|файл-приёмник|
|`bs=`|размер блока (байт), влияет на скорость|
|`count=`|сколько блоков скопировать|
|`skip=`|пропустить N блоков в источнике|
|`seek=`|пропустить N блоков в приёмнике|
|`conv=`|преобразования (`noerror`, `sync`, `notrunc`)|
|`status=`|вывод прогресса (`none`, `noxfer`, `progress`)|
Примеры:
``` bash
dd if=/dev/sda of=/tmp/disk.img bs=1M count=100       # копировать 100 МБ с диска
dd if=/tmp/disk.img of=/dev/sdb bs=1M                 # записать образ на диск
dd if=/dev/zero of=file.img bs=1M count=10            # создать файл 10 МБ, заполненный нулями
dd if=/dev/urandom of=file.bin bs=1M count=1          # создать файл 1 МБ с случайными данными
dd if=/dev/sda of=/dev/null bs=1M status=progress     # протестировать скорость чтения
dd if=file1 of=file2 conv=notrunc,noerror             # копирование без обрезки, игнорируя ошибки
```
