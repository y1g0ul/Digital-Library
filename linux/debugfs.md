Утилита в [[Linux]] для низкоуровневой работы с файловой системой **`ext2/ext3/ext4`**

``` bash
debugfs [опции] устройство
```

|Опция|Описание|
|---|---|
|`-w`|открыть файловую систему в режиме записи|
|`-R`|выполнить одну команду и выйти|
|`-f`|выполнить команды из файла|
|`-V`|показать версию программы|
Основные команды внутри debugfs

|Команда|Описание|
|---|---|
|`ls`|показать файлы в каталоге|
|`cd`|перейти в каталог|
|`stat`|показать информацию об inode|
|`cat`|вывести содержимое файла|
|`dump`|извлечь файл из файловой системы|
|`rm`|удалить файл|
|`quit`|выйти|
``` bash
sudo debugfs /dev/sdb1                     # открыть файловую систему

sudo debugfs -R "ls /" /dev/sdb1           # показать содержимое корня

sudo debugfs -R "stat /file.txt" /dev/sdb1 # информация об inode файла

sudo debugfs -R "cat /file.txt" /dev/sdb1  # вывести содержимое файла

sudo debugfs -R "dump /file.txt file.txt" /dev/sdb1  # извлечь файл из ФС

sudo debugfs -w /dev/sdb1                  # открыть ФС с возможностью записи
```
