Утилита в [[Linux]] для **поиска строк по шаблону** в файлах или стандартном вводе. Поддерживает регулярные выражения.

``` bash
grep [опции] шаблон [файл]...
```

|Опция|Описание|
|---|---|
|`-i`|Игнорировать регистр|
|`-v`|Инвертировать поиск (показать НЕ совпадающие строки)|
|`-r`|Рекурсивный поиск по каталогам|
|`-n`|Показать номер строки|
|`-l`|Показать только имена файлов|
|`-c`|Подсчитать количество совпадений|
|`-w`|Искать целое слово|
|`-x`|Полное совпадение строки|
|`-E`|Расширенные регулярные выражения|
|`-F`|Поиск фиксированной строки (без regex)|
|`-A N`|Показать N строк после совпадения|
|`-B N`|Показать N строк до совпадения|
|`-C N`|Показать N строк контекста|
``` bash
grep root /etc/passwd              # поиск строки "root"
grep -i error logfile.txt          # без учёта регистра
grep -r "main" ./src               # рекурсивный поиск
grep -n bash /etc/passwd           # показать номера строк
grep -v "^#" config.conf           # исключить комментарии
grep -c sshd /var/log/auth.log     # подсчитать совпадения
grep -E "cat|dog" animals.txt      # расширенное регулярное выражение
ps aux | grep nginx                # поиск процесса
