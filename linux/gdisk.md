Утилита [[Linux]] для управления таблицей разделов GPT на дисках

``` bash
gdisk диск
```

|Команда|Описание|
|---|---|
|`p`|Показать таблицу разделов|
|`n`|Создать новый раздел|
|`d`|Удалить раздел|
|`c`|Изменить имя раздела|
|`t`|Изменить тип раздела|
|`w`|Записать изменения на диск|
|`q`|Выйти без сохранения|
|`?`|Показать справку|
``` bash
sudo gdisk /dev/sdb
p      # посмотреть текущие разделы
n      # создать новый раздел
w      # сохранить изменения
```
