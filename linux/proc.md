Это иерархическое представление всех процессов в системе [[Linux]].
Всё в `/proc` — **не настоящие файлы**,они создаются ядром и **меняются в реальном времени**.

Числовые каталоги (`/proc/1234`)
- Каждый номер = [[PID]].
- Информация о конкретном [[процесс]]е
Примеры:
- `cmdline` — команда запуска
- `status` — состояние процесса
- `fd/` — открытые файлы

Основные системные файлы
- **`/proc/cpuinfo`** — информация о CPU
- **`/proc/meminfo`** — использование памяти
- **`/proc/uptime`** — сколько система работает
- **`/proc/loadavg`** — нагрузка
- **`/proc/version`** — версия ядра

Ядро и настройки
- **`/proc/sys/`** — параметры ядра (sysctl)
- **`/proc/interrupts`** — прерывания
- **`/proc/modules`** — загруженные модули

Сеть
- **`/proc/net/`** — состояние сети
- TCP/UDP соединения
- маршруты
