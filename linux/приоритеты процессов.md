В [[Linux]] каждому процессу назначается **приоритет выполнения**, который влияет на распределение CPU.

Приоритет управляется через значение **nice**.  
Чем меньше значение nice → тем выше приоритет процесса.

Диапазон nice: **от -20 (высокий приоритет) до 19 (низкий приоритет)**.

``` bash
nice [опции] команда
renice приоритет PID
```

| Команда / параметр   | Описание                               |
| -------------------- | -------------------------------------- |
| `nice -n N команда`  | Запустить процесс с заданным nice      |
| `renice N PID`       | Изменить nice у существующего процесса |
| [[top]]              | Просмотр значения NI                   |
| [[ps]] -o pid,ni,cmd | Показать nice процессов                |
``` bash
nice -n 10 sleep 100         # запустить с пониженным приоритетом
renice 15 1234               # изменить приоритет PID 1234
ps -o pid,ni,cmd             # посмотреть nice процессов
top                          # смотреть столбец NI
```
