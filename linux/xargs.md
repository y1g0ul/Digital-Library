Утилита в [[Linux]] для **построения аргументов команд из стандартного ввода**.  
Берёт вывод одной команды и передаёт его как аргументы другой.

``` bash
xargs [опции] команда
```

|Опция|Описание|
|---|---|
|`-n <N>`|Передавать не более N аргументов за один запуск|
|`-I {}`|Шаблон подстановки аргументов|
|`-0`|Использовать `\0` как разделитель (для `find -print0`)|
|`-r`|Не запускать команду, если вход пустой|
|`-p`|Запрашивать подтверждение перед выполнением|
|`-t`|Печатать команду перед выполнением|
|`-L <N>`|Использовать N строк на один запуск|
|`-P <N>`|Параллельное выполнение|
Примеры:
``` bash
ls *.log | xargs rm                     # удалить все .log файлы
find . -name "*.tmp" | xargs rm         # удалить найденные файлы
find . -name "*.tmp" -print0 | xargs -0 rm  # безопасно для пробелов
cat users.txt | xargs echo              # передать строки как аргументы
echo 1 2 3 | xargs -n 1 echo             # по одному аргументу за запуск
cat files.txt | xargs -I {} cp {} /tmp   # подстановка аргумента
```
