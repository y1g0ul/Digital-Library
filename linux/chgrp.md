Команда в [[Linux]] которая меняет группу файла. Может использоваться владельцем файла, если он состоит в целевой группе.

``` bash
chgrp [опции] группа файл
```

|Опция|Описание|
|---|---|
|`-R`|рекурсивно изменить для каталога|
|`-v`|показать изменения|
|`-c`|показать только изменённые файлы|
|`--reference=FILE`|копировать группу с другого файла|
``` bash
chgrp group file.txt                 # изменить группу файла

chgrp -R group dir/                  # изменить группу рекурсивно

chgrp --reference=file1 file2        # копировать группу с другого файла

chgrp -v group file.txt              # показать изменения
```
