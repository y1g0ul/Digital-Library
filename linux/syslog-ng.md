Служба в [[Linux]] для сбора, фильтрации, обработки и пересылки журналов. Задуман как централизированная система сбора логов в локальной сети, но может работать и как локальная система логирования. 
``` bash
syslog-ng
```

| Ключ                | Что делает                          |
| ------------------- | ----------------------------------- |
| `-s`                | проверить конфигурацию              |
| `-F`                | не переходить в фоновый режим       |
| `-d`                | отладочный режим                    |
| `-v`                | показать версию                     |
| `-f`                | использовать указанный конфиг       |
| `--persist-file`    | указать файл состояния              |
| `--preprocess-into` | сгенерировать итоговую конфигурацию |
