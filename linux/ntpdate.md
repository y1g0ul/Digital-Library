Команда в [[Linux]] для единоразовой синхронизации системного времени с NTP-сервером.
``` bash
ntpdate [опции] NTP-сервер
```

| Ключ | Что делает                                   |
| ---- | -------------------------------------------- |
| `-q` | только запросить время без изменения системы |
| `-u` | использовать непривилегированный порт        |
| `-b` | резко установить время (step)                |
| `-d` | отладочный режим                             |
| `-s` | вывод через syslog                           |
| `-v` | подробный вывод                              |
``` bash
sudo ntpdate pool.ntp.org     # синхронизировать время

sudo ntpdate time.google.com  # использовать сервер Google

ntpdate -q pool.ntp.org       # только проверить разницу

sudo ntpdate -u pool.ntp.org  # использовать непривилегированный порт

sudo ntpdate -b pool.ntp.org  # принудительно установить время

ntpdate -d pool.ntp.org       # отладка
```
