Команда в [[Linux]] для управления пользовательскими заданиями [[cron]].
``` bash
crontab [опции]
```

| Ключ | Что делает                                     |
| ---- | ---------------------------------------------- |
| `-e` | редактировать cron-задачи                      |
| `-l` | показать текущие cron-задачи                   |
| `-r` | удалить все cron-задачи                        |
| `-i` | запросить подтверждение перед удалением        |
| `-u` | работать с cron указанного пользователя (root) |
``` bash
crontab -e                    # открыть cron на редактирование

crontab -l                    # показать текущие задания

crontab -r                    # удалить все задания

crontab -i -r                 # удалить с подтверждением

sudo crontab -u user -l       # посмотреть задания пользователя

sudo crontab -u user -e       # изменить задания пользователя
```
