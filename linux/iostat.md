Команда которая показывает статистику CPU и устройств ввода-вывода (диски).

``` bash
iostat [ключи] [интервал]
```

| Ключ | Что делает                        |
| ---- | --------------------------------- |
| `-c` | только статистика CPU             |
| `-d` | только статистика дисков          |
| `-x` | расширенная статистика устройств  |
| `-m` | вывод в MB вместо KB              |
| `-p` | показать разделы дисков           |
| `-y` | убрать первый отчёт после запуска |
Основные поля CPU:

| Поле      | Что показывает                                           |
| --------- | -------------------------------------------------------- |
| `%user`   | время работы пользовательских процессов                  |
| `%nice`   | время работы процессов с изменённым приоритетом [[nice]] |
| `%system` | работа ядра [[Linux]]                                    |
| `%iowait` | ожидание завершения операций ввода-вывода                |
| `%steal`  | время, забранное гипервизором (виртуальные машины)       |
| `%idle`   | простой CPU                                              |
Поля устройств:

| Поле        | Что показывает                             |
| ----------- | ------------------------------------------ |
| `tps`       | количество операций ввода-вывода в секунду |
| `kB_read/s` | скорость чтения данных                     |
| `kB_wrtn/s` | скорость записи данных                     |
| `kB_dscd/s` | скорость удаления данных (discard/TRIM)    |
| `kB_read`   | общее количество прочитанных данных        |
| `kB_wrtn`   | общее количество записанных данных         |
Примеры:
``` bash
iostat                  # показать статистику CPU и дисков
iostat 2                # обновлять каждые 2 секунды
iostat -x               # расширенная статистика дисков
iostat -xm 1            # расширенная статистика в MB каждую секунду
iostat -c               # показать только CPU
iostat -d               # показать только диски
iostat -p               # показать разделы дисков
```
