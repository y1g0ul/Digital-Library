Команда  которая показывает статистику загрузки процессора и устройств ввода-вывода (дисков). Используется для диагностики производительности системы: позволяет определить проблемы с CPU, дисками и задержками операций ввода-вывода.

```bash
iostat <ключи> <интервал> <количество>
```

| Ключ | Описание |
| ---- | -------- |
| `-c` | показать только статистику CPU |
| `-d` | показать только статистику устройств |
| `-x` | показать расширенную статистику устройств |
| `-m` | выводить значения в мегабайтах |
| `-h` | удобный для чтения формат |
| `-y` | пропустить первый отчёт после запуска |
| `-p` | показать статистику разделов диска |

---

## Базовое использование

```bash
iostat
```

Показывает:

- загрузку CPU
- статистику операций дисков

---

## Мониторинг в реальном времени

```bash
iostat 2
```

Обновляет информацию каждые 2 секунды.

```bash
iostat 2 5
```

Показывает статистику каждые 2 секунды 5 раз.

---

## Расширенная информация о дисках

```bash
iostat -x
```

Показывает дополнительные параметры:

| Поле | Значение |
| ---- | -------- |
| `r/s` | количество операций чтения в секунду |
| `w/s` | количество операций записи в секунду |
| `rkB/s` | скорость чтения данных |
| `wkB/s` | скорость записи данных |
| `await` | среднее время ожидания операции |
| `%util` | процент времени, когда диск был занят |

---

## Вывод в мегабайтах

```bash
iostat -xm
```

`-x` — расширенная статистика  
`-m` — значения в MB вместо KB

---

## Только CPU

```bash
iostat -c
```

Пример вывода:

```text
avg-cpu:
%user   %system   %iowait   %idle
```

| Поле | Значение |
| ---- | -------- |
| `%user` | время работы пользовательских процессов |
| `%system` | время работы ядра |
| `%iowait` | время ожидания операций ввода-вывода |
| `%idle` | время простоя CPU |

---

## Только диски

```bash
iostat -d
```

Показывает статистику устройств хранения.

---

## Показать разделы диска

```bash
iostat -p
```

Выводит информацию не только по диску, но и по его разделам.

---

## Полезные комбинации

```bash
iostat -xm 2
```

Расширенная статистика дисков с обновлением каждые 2 секунды.

```bash
iostat -xy 1
```

Расширенная статистика без первого отчёта.

---

## Анализ проблем

Высокий `%iowait`:

```text
%iowait = 40%
```

означает, что CPU часто ждёт завершения операций ввода-вывода.

Возможные причины:

- медленный диск
- высокая нагрузка на накопитель
- большое количество операций чтения/записи

---

Высокий `%util`:

```text
%util = 100%
```

означает, что устройство постоянно занято.

---

Большой `await`:

```text
await = 150ms
```

означает высокие задержки операций диска.

---

## Примеры

```bash
iostat -x 5
# показывает расширенную статистику дисков каждые 5 секунд

iostat -xm 1
# показывает расширенную статистику в MB каждую секунду

iostat -c
# показывает только нагрузку CPU

iostat -d
# показывает только устройства хранения
```
