# iostat

Команда которая показывает статистику CPU и устройств ввода-вывода (диски).

``` bash
iostat [ключи] [интервал]
```

|Ключ|Что делает|
|---|---|
|`-c`|только статистика CPU|
|`-d`|только статистика дисков|
|`-x`|расширенная статистика устройств|
|`-m`|вывод в MB вместо KB|
|`-p`|показать разделы дисков|
|`-y`|убрать первый отчёт после запуска|

Основные поля:

| Поле      | Что показывает                                |
| --------- | --------------------------------------------- |
| `%user`   | нагрузка CPU пользовательскими [[процесс]]ами |
| `%system` | нагрузка CPU ядром                            |
| `%iowait` | время ожидания операций ввода-вывода          |
| `r/s`     | операции чтения в секунду                     |
| `w/s`     | операции записи в секунду                     |
| `await`   | задержка операций диска                       |
| `%util`   | занятость устройства                          |

Примеры:
``` bash
iostat                    # показать общую статистику
iostat 2                  # обновлять каждые 2 секунды
iostat -x                # расширенная информация о дисках
iostat -xm 1             # расширенная статистика в MB каждую секунду
iostat -c                # показать только CPU
iostat -d                # показать только диски
iostat -xy 2             # расширенная статистика без первого отчёта
```
