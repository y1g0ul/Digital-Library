Показывает размер файлов и каталогов на диске.
Часто используется вместе с [[df]], чтобы понять, что именно занимает место.

``` bash
du [опции] [файл или каталог]

если файл/каталог не указан → по текущей директории
```

|Опция|Что делает|
|---|---|
|`-h`|человеко-читаемый формат (KB, MB, GB)|
|`-H`|человеко-читаемый, основание 1000|
|`-s`|суммарный размер (не выводить для подкаталогов)|
|`-a`|показывать все файлы, не только каталоги|
|`-c`|показывать общий итог в конце|
|`--max-depth=N`|показать только N уровней вложенности|
|`-x`|ограничить вывод одной файловой системой|
|`--apparent-size`|показать фактический размер файлов (без учёта блоков)|
Примеры:
``` bash 
du -h                   # человеко-читаемый размер всех каталогов
du -sh                  # суммарный размер текущей директории
du -ah                  # все файлы и каталоги с размером
du -h --max-depth=1      # размер подкаталогов только первого уровня
du -csh *               # суммарный размер всех элементов + итог
du -x -h /home           # размер /home только на одной FS
du --apparent-size -h file.txt  # показать фактический размер файла
```
