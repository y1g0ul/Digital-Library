Утилита в [[Linux]] для вывода информации о файловых системах **`ext2/ext3/ext4`**


``` bash
dumpe2fs [опции] устройство
```

Команда выводит:
- размер блока
- количество блоков
- количество inode
- время последней проверки    
- время последнего монтирования    
- состояние файловой системы
- резервные суперблоки

|Опция|Описание|
|---|---|
|`-h`|показать только информацию из суперблока|
|`-b`|показать повреждённые блоки|
|`-f`|принудительно выполнить команду|
|`-x`|выводить номера блоков в шестнадцатеричном виде|
