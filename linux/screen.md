Утилита в [[Linux]] для создания **виртуальных терминальных сессий**, которые продолжают работать после закрытия SSH или терминала. Позволяет отсоединяться (detach) от сессии и подключаться к ней позже.

`screen [опции]`
Для подключения к существующей сессии:
`screen -r [имя_или_ID]`

|Опция|Описание|
|---|---|
|`-S имя`|Создать сессию с именем|
|`-r`|Подключиться к существующей сессии|
|`-ls`|Показать список сессий|
|`-d`|Отсоединить сессию|
|`-d -r`|Переподключиться к сессии|
|`-X`|Выполнить команду в сессии|

Основные сочетания клавиш (внутри screen)

|Комбинация|Действие|
|---|---|
|`Ctrl+A d`|Отсоединиться от сессии|
|`Ctrl+A c`|Создать новое окно|
|`Ctrl+A n`|Следующее окно|
|`Ctrl+A p`|Предыдущее окно|
|`Ctrl+A "`|Список окон|
|`Ctrl+A k`|Закрыть окно|

``` bash
screen                 # создать новую сессию
screen -S work         # создать сессию с именем work
screen -ls             # список сессий
screen -r work         # подключиться к сессии work
screen -d -r work      # принудительно переподключиться
```
