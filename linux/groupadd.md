Команда в [[Linux]] для создания новой группы
``` bash
groupadd [опции] группа
```

| Ключ | Что делает                                     |
| ---- | ---------------------------------------------- |
| `-g` | задать GID (ID группы)                         |
| `-f` | не выдавать ошибку, если группа уже существует |
| `-r` | создать системную группу                       |
| `-K` | переопределить значения из `/etc/login.defs`   |
``` bash
groupadd developers           # создать группу

groupadd -g 2000 devs         # задать GID

groupadd -r service_group     # системная группа

groupadd -f devs              # не ошибка, если уже есть
```
