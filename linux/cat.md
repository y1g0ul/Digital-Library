Команда в [[Linux]] для вывода содержимого файлов на экран и объединения файлов
``` bash
cat [опции] файл
```

|Ключ|Что делает|
|---|---|
|`-n`|нумеровать все строки|
|`-b`|нумеровать только непустые строки|
|`-s`|сжимать несколько пустых строк в одну|
|`-E`|показывать конец строки символом `$`|
|`-T`|показывать символы табуляции как `^I`|
|`-A`|показывать все непечатаемые символы|
|`-v`|отображать непечатаемые символы|
``` bash
cat file.txt                  # вывести содержимое файла

cat file1.txt file2.txt       # объединить вывод файлов

cat -n file.txt               # показать номера строк

cat -b file.txt               # номера только непустых строк

cat -E file.txt               # показать окончания строк

cat -T file.txt               # показать табуляции

cat -A file.txt               # показать все спецсимволы

cat > file.txt                # создать файл и записать текст

cat >> file.txt               # добавить текст в конец файла
```
