Команда в [[Linux]] для запуска всех исполняемых файлов из каталога.
``` bash
run-parts [опции] каталог
```

| Ключ        | Что делает                                              |
| ----------- | ------------------------------------------------------- |
| `--test`    | показать какие скрипты будут запущены                   |
| `--verbose` | подробный вывод                                         |
| `--report`  | выводить информацию только если скрипт что-то напечатал |
| `--reverse` | запускать в обратном порядке                            |
``` bash
run-parts /etc/cron.daily

run-parts --test /etc/cron.daily

run-parts --verbose /etc/cron.weekly
```
