В [[Linux]] специальные символы позволяют выбирать группы файлов по шаблону:

Основные шаблоны
- `*` — любой набор символов (включая пустой).  
    Пример: `rm S*` — удалить все файлы, начинающиеся на **S**.
- `?` — ровно один любой символ.  
    Пример: `rm S?` — удалить файлы, где **S** + **1 символ**.
- `??` и т.д. — каждый `?` соответствует одному символу.  
    Пример: `rm S??` — удалить файлы, где **S** + **2 символа**.
Дополнительно
- `[abc]` — любой из перечисленных символов.
- `[a-z]` — любой символ из диапазона.
