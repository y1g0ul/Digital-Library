Для файлов — `644`, для директорий — `755`.
Берём некую маску, которую вычитаем из `666` для файлов и `777` для директорий. По умолчанию это `022`. Таким образом, `666` – `022` = `644`, `777` – `022` = `755`. Задается командой  `umask 022`

утилита в [[Linux]] для настройки прав доступа по умолчанию при создании файлов и каталогов.
`umask [значение]`

Базовые права:

|Тип|Значение|
|---|---|
|файл|`666` (rw-rw-rw-)|
|каталог|`777` (rwxrwxrwx)|
`umask` вычитает права из этих значений
