утилита в [[Linux]] для настройки прав доступа по умолчанию при создании файлов и каталогов. Она не задаёт права напрямую, а убирает (mask) из их базовых значений.
```
umask [значение]
```

Базовые права:

|Тип|Значение|
|---|---|
|файл|`666` (rw-rw-rw-)|
|каталог|`777` (rwxrwxrwx)|
**`USERGROUPS_ENAB`**
Если включено:
- `USERGROUPS_ENAB yes`
то:
- umask 022 → фактически становится 002
Это происходит, если у пользователя есть своя группа (user:user)

``` bash
umask                 # посмотреть текущий umask

umask 022             # стандартные права (644/755)

umask 002             # совместная работа (664/775)

umask 077             # только владелец (600/700)

umask -S              # показать в символьном виде
```
