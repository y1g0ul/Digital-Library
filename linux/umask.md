утилита в [[Linux]] для настройки прав доступа по умолчанию при создании файлов и каталогов. Она не задаёт права напрямую, а убирает (mask) их из базовых значений.
```
umask [значение]
```

Базовые права:

|Тип|Значение|
|---|---|
|файл|`666` (rw-rw-rw-)|
|каталог|`777` (rwxrwxrwx)|


``` bash
umask                    # показать текущую маску

umask 022                # установить стандартную маску

umask 077                # только владелец имеет доступ

umask -S                 # показать в символьном виде
```
