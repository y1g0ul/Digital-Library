Встроенная команда оболочки ([[Linux]]) для просмотра фоновых и приостановленных задач текущей сессии shell.

jobs [опции]

|Опция|Описание|
|---|---|
|`-l`|Показать PID процессов|
|`-p`|Показать только PID|
|`-r`|Только запущенные (running)|
|`-s`|Только остановленные (stopped)|
``` bash
sleep 100 &          # запустить задачу в фоне
jobs                 # показать список задач
jobs -l              # показать задачи с PID
jobs -r              # только выполняющиеся задачи
jobs -s              # только остановленные задачи
```
