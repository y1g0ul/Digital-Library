Утилита в [[Linux]] для поиска процессов по имени, пользователю и другим параметрам. Выводит [[PID]] найденных процессов. Удобная альтернатива связке [[ps]] | [[grep]].

`pgrep [опции] шаблон`
`шаблон` — имя процесса или регулярное выражение.

|Опция|Описание|
|---|---|
|`-l`|Показать имя процесса вместе с PID|
|`-a`|Показать полную командную строку|
|`-u user`|Процессы указанного пользователя|
|`-x`|Точное совпадение имени|
|`-f`|Искать по всей командной строке|
|`-n`|Только самый новый процесс|
|`-o`|Только самый старый процесс|
|`-c`|Показать количество найденных процессов|
``` bash
pgrep nginx              # найти PID процессов nginx
pgrep -l nginx           # PID + имя процесса
pgrep -u user            # процессы пользователя user
pgrep -f python          # поиск по всей командной строке
pgrep -x sshd            # точное совпадение имени
pgrep -c bash            # количество процессов bash
```
