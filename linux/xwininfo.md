Команда в [[Linux]] (X11) для получения информации об окнах
``` bash
xwininfo [опции]
```

| Ключ      | Что делает                             |
| --------- | -------------------------------------- |
| `-root`   | показать информацию о корневом окне    |
| `-tree`   | показать дерево окон                   |
| `-int`    | использовать числовые ID окон          |
| `-id`     | указать ID окна                        |
| `-name`   | искать окно по имени                   |
| `-stats`  | показать дополнительные характеристики |
| `-bits`   | информация о битах/глубине цвета       |
| `-events` | показать события окна                  |
| `-size`   | информация о размере                   |
| `-shape`  | информация о форме окна                |
