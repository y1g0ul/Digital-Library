утилита в [[Linux]] для быстрого поиска файлов по имени.

В отличии от [[find]] работает не напрямую с файловой системой, а с предварительно созданной базой данных, поэтому ищет очень быстроты.

``` bash
locate [опции] имя
```

|Опция|Описание|
|---|---|
|`-i`|игнорировать регистр|
|`-r`|поиск по регулярному выражению|
|`-c`|вывести только количество совпадений|
|`-n N`|ограничить число результатов|
|`-e`|показывать только существующие файлы|
|`-l N`|ограничить вывод (аналог `-n`)|
`locate` использует базу файлов, которая создаётся командой `updatedb`. Конфиг распологается по пути `/etc/updatedb.conf `.

``` bash
locate file.txt             # найти файл по имени

locate -i file              # поиск без учёта регистра

locate -c file              # количество совпадений

locate -n 5 file            # показать первые 5 результатов

locate -r "\.log$"          # поиск по regex (файлы .log)

locate -e file              # только существующие файлы

sudo updatedb               # обновить базу данных
```
