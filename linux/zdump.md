Команда в [[Linux]] для просмотра информации о часовых поясах и переходах на летнее/зимнее время.

| Ключ        | Что делает                             |
| ----------- | -------------------------------------- |
| `-v`        | подробный вывод всех переходов времени |
| `-c`        | ограничить диапазон лет для вывода     |
| `-V`        | подробный вывод без крайних значений   |
| `-i`        | краткая информация о переходах         |
| `--help`    | справка по команде                     |
| `--version` | версия программы                       |
``` bash
zdump Europe/Moscow           # информация о часовом поясе

zdump Asia/Singapore          # информация о часовом поясе

zdump -v Europe/London        # все переходы DST

zdump -c 2020,2030 -v Europe/Berlin
# переходы с 2020 по 2030 год

zdump -i Europe/London        # краткий список переходов

zdump -v America/New_York | less
# подробная история изменений
```
