Команда в [[Linux]] для просмотра сокетов и сетевых соединений

``` bash
ss [опции]
```

|Ключ|Что делает|
|---|---|
|`-t`|TCP соединения|
|`-u`|UDP соединения|
|`-l`|только слушающие сокеты|
|`-n`|не преобразовывать адреса и порты|
|`-p`|показать процессы|
|`-a`|все сокеты|
|`-s`|общая статистика|
|`-r`|показать DNS/host имена|
|`-4`|только IPv4|
|`-6`|только IPv6|
|`-c`|обновлять вывод постоянно|
``` bash
ss                              # все соединения
ss -tuln                        # слушающие TCP/UDP порты
ss -tulpn                       # порты + процессы
ss -an                          # все сокеты без DNS
ss -s                           # статистика сокетов
ss -4                           # только IPv4
ss -6                           # только IPv6

ss -tulnp | grep 80             # кто слушает 80 порт
```

> [!info]  инфо
> ss работает быстрее [[netstat]], так как получает информацию напрямую из kernel space.
