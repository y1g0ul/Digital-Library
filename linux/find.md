Команда в [[Linux]] которая ищет файлы и каталоги
``` bash
find [где искать] [условия] [действия]
```

|Ключ|Что делает|
|---|---|
|`-name`|поиск по имени (с учётом регистра)|
|`-iname`|поиск по имени без учёта регистра|
|`-type f`|только файлы|
|`-type d`|только каталоги|
|`-size`|размер файла|
|`-mtime`|изменение по времени (дни)|
|`-user`|владелец файла|
|`-perm`|права доступа|
|`-maxdepth`|глубина поиска|
|`-exec`|выполнить команду|
|`-delete`|удалить найденное|
Примеры:
``` bash
find . -name "file.txt"              # найти файл по имени
find /etc -iname "*.conf"            # найти .conf без учёта регистра
find . -type f                       # найти все файлы
find . -type d                       # найти все каталоги
find . -size +100M                   # найти файлы больше 100 МБ
find . -mtime -1                     # файлы, изменённые за последний день
find . -user root                    # файлы пользователя root
find . -perm 644                     # файлы с правами 644
find . -maxdepth 1 -type f           # искать только в текущем каталоге
find . -name "*.log" -exec rm {} \;  # удалить найденные .log
```
