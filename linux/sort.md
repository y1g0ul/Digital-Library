Команда в [[Linux]] которая упорядочивает строки текстовых файлов или потоков данных (по алфавиту, числам или в обратном порядке) и выводит результат в стандартный вывод.
``` bash
sort [опции] [файл]...
```

|Опция|Описание|
|---|---|
|`-n`|Числовая сортировка|
|`-r`|Сортировка в обратном порядке|
|`-u`|Удалить дубликаты (уникальные строки)|
|`-k N`|Сортировка по N-му полю (колонке)|
|`-t CHAR`|Указать разделитель полей|
|`-h`|Сортировка чисел с суффиксами (`K`, `M`, `G`)|
|`-V`|«Версионная» сортировка (`1.9 < 1.10`)|
|`-o FILE`|Записать результат в файл|
|`-f`|Игнорировать регистр|
Примеры:
``` bash
sort file.txt                      # Алфавитная сортировка строк
sort -n numbers.txt                # Числовая сортировка
sort -r file.txt                   # Обратный порядок
sort -u file.txt                   # Удалить дубликаты
sort -k 2 users.txt                # Сортировка по 2 колонке
sort -t ':' -k 3 /etc/passwd       # Сортировка по UID
du -h | sort -h                    # Сортировка по размеру
ps aux | sort -k 3 -n              # Сортировка процессов по CPU
```
