Утилита в  [[Linux]] для создания, удаленя и изменения разделов диска. В сравнении с [[fdisk]] используется в скриптах. 

``` bash
parted [диск]
```

| Команда      | Описание                       |
| ------------ | ------------------------------ |
| `print`      | Показать таблицу разделов      |
| `mklabel`    | Создать новую таблицу разделов |
| `mkpart`     | Создать новый раздел           |
| `rm`         | Удалить раздел                 |
| `resizepart` | Изменить размер раздела        |
| `quit`       | Выйти                          |
