Команда в [[Linux]] для запуска демона NTP, обеспечивающего постоянную синхронизацию системного времени
``` bash
ntpd [опции]
```

| Ключ | Что делает                                             |
| ---- | ------------------------------------------------------ |
| `-g` | разрешить большую первоначальную корректировку времени |
| `-q` | однократная синхронизация и завершение работы          |
| `-d` | отладочный режим                                       |
| `-n` | не переходить в фоновый режим                          |
| `-u` | запуск от указанного пользователя                      |
| `-p` | использовать указанный PID-файл                        |
| `-c` | указать конфигурационный файл                          |
``` bash
sudo ntpd                    # запустить демон

sudo ntpd -g                 # разрешить большую коррекцию времени

sudo ntpd -q                 # однократная синхронизация

sudo ntpd -gq                # синхронизация и завершение

sudo ntpd -d                 # отладочный режим

sudo ntpd -c /etc/ntp.conf   # использовать свой конфиг
```
