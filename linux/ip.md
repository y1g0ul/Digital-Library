Команда в [[Linux]] для управления сетевыми интерфейсами, IP-адресами, маршрутами, ARP-таблицей и другими параметрами сети. Входит в пакет **iproute2** и является современной заменой `ifconfig`, `route` и `arp`.
``` bash
ip [объект] [команда] [параметры]
```

|Объект|Что делает|
|---|---|
|`addr` (`a`)|управление IP-адресами|
|`link` (`l`)|управление сетевыми интерфейсами|
|`route` (`r`)|просмотр и изменение маршрутов|
|`neigh` (`n`)|просмотр и управление ARP/NDP-кэшем|
|`rule`|правила маршрутизации|
|`monitor`|мониторинг изменений сети|
|`tunnel`|управление туннелями|
|`netns`|управление сетевыми пространствами имён|
**`Работа с IP-адресами`**
``` bash
ip addr show # Показать все адреса
ip addr show eth0 # Показать адрес интерфейса
sudo ip addr add 192.168.1.100/24 dev eth0 # Добавить IP
sudo ip addr del 192.168.1.100/24 dev eth0 # Удалить IP
```
