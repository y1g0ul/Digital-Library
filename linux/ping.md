Команда в [[Linux]] для проверки доступности узла и измерения времени передачи пакетов по сети с помощью протокола [[ICMP]].

``` bash
ping [ключи] <хост>
```

| Ключ | Что делает |
|------|------------|
| `-c` | количество отправляемых пакетов |
| `-i` | интервал между пакетами (сек.) |
| `-W` | время ожидания ответа (сек.) |
| `-s` | размер ICMP-пакета |
| `-4` | использовать только IPv4 |
| `-6` | использовать только IPv6 |

Примеры:

``` bash
ping 8.8.8.8                  # проверить доступность узла
ping google.com               # проверить DNS и доступность
ping -c 4 8.8.8.8             # отправить 4 пакета
ping -i 2 8.8.8.8             # интервал между пакетами 2 секунды
ping -W 1 8.8.8.8             # ждать ответ не более 1 секунды
ping -s 1000 8.8.8.8          # отправить пакет размером 1000 байт
ping -4 google.com            # использовать IPv4
ping -6 google.com            # использовать IPv6
```
