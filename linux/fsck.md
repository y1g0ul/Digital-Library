File System Check - утилита в [[Linux]] для проверки и исправления файловых систем.

``` bash 
fsck [опции] устройство
```

|Опция|Описание|
|---|---|
|`-A`|проверить все файловые системы из `/etc/fstab`|
|`-N`|показать что будет выполнено, без запуска|
|`-R`|не проверять root файловую систему|
|`-T`|не показывать заголовок при запуске|
|`-V`|подробный вывод|
|`-y`|автоматически отвечать "yes" на исправления|
|`-n`|не исправлять ошибки (только проверка)|
