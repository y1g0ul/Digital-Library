File System Check - утилита в [[Linux]] для проверки и исправления файловых систем.

``` bash 
fsck [опции] устройство
```

|Опция|Описание|
|---|---|
|`-A`|проверить все файловые системы из `/etc/fstab`|
|`-N`|показать что будет выполнено, без запуска|
|`-R`|не проверять root файловую систему|
|`-T`|не показывать заголовок при запуске|
|`-V`|подробный вывод|
|`-y`|автоматически отвечать "yes" на исправления|
|`-n`|не исправлять ошибки (только проверка)|
``` bash
sudo fsck /dev/sdb1                     # проверить файловую систему

sudo fsck -y /dev/sdb1                  # автоматически исправить ошибки

sudo fsck -n /dev/sdb1                  # только проверка без исправления

sudo fsck -A                            # проверить все файловые системы

sudo umount /dev/sdb1 && sudo fsck /dev/sdb1   # размонтировать и проверить
```

`fsck` - это обертка, которая запускает проверку конкретной файловой системы.
