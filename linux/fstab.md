Конфигурационный файл в [[Linux]], который хранится по пути `/etc/fstab` и отвечает за **автоматическое** подключение файловых систем при загрузке.

Что в нём хранится:
- Устройства (диски, разделы, сетевые хранилища)
- Точки монтирования (куда будет подключён раздел, например `/`, `/home`)
- Тип файловой системы (например `ext4`, `ntfs`)
- Опции монтирования (например `ro` — только чтение, `defaults`)
- Параметры для проверки файловой системы при загрузке
