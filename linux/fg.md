foreground - команда в [[Linux]] которая переводит приостановленные или работающие в фоновом режиме [[процесс]]ы обратно на передний план, возвращая им управление термналом.

`fg %номер задачи` [[jobs]]

``` bash
sleep 200 &      # запустить в фоне
jobs             # посмотреть номер задачи
fg %1            # вернуть задачу №1 в передний план
Ctrl+Z           # приостановить текущий процесс
fg               # вернуть последнюю задачу
```
