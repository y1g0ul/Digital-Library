Утилита в [[Linux]] (Ubuntu) для настройки сетевых интерфейсов с помощью YAML-конфигурации. 

``` bash
netplan <команда>
```

| Команда | Что делает |
|---------|------------|
| `generate` | сгенерировать конфигурацию для backend |
| `apply` | применить конфигурацию |
| `try` | временно применить конфигурацию с возможностью отката |
| `get` | показать текущую конфигурацию |
| `set` | изменить параметр конфигурации |
| `status` | показать состояние интерфейсов |
| `ip` | показать IP-адреса интерфейсов |

Конфигурационные файлы:

```text
/etc/netplan/*.yaml
```

Backend (кто применяет настройки):

| Backend | Назначение |
|---------|------------|
| `systemd-networkd` | серверы |
| `NetworkManager` | настольные системы |

Примеры:

``` bash
netplan generate                 # сгенерировать конфигурацию
netplan apply                    # применить настройки
netplan try                      # применить с возможностью отката
netplan status                   # показать состояние сети
netplan get                      # показать текущую конфигурацию
netplan ip                       # показать IP-адреса
```

Пример статической настройки:

```yaml
network:
  version: 2
  ethernets:
    enp0s3:
      dhcp4: false
      addresses:
        - 192.168.1.100/24
      routes:
        - to: default
          via: 192.168.1.1
      nameservers:
        addresses:
          - 8.8.8.8
          - 1.1.1.1
```
