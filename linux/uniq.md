Удаление повторяющихся строк в [[Linux]]

``` bash
uniq [опции] файл
	или
sort файл | uniq
```

`uniq`:
- **не ищет одинаковые строки по всему файлу**
- он сравнивает **только строки, идущие подряд**

| Опция    | Что делает простыми словами                                |
| -------- | ---------------------------------------------------------- |
| `-c`     | Показывает, **сколько раз строка повторяется подряд**      |
| `-d`     | Показывает **только строки, которые повторялись**          |
| `-u`     | Показывает **только строки, которые встретились один раз** |
| `-i`     | Игнорирует регистр (`Apple` = `apple`)                     |
| `-f N`   | **Игнорирует первые N полей** (разделённых пробелами)      |
| `-s N`   | **Игнорирует первые N символов строки**                    |
| `-w N`   | Сравнивает **только первые N символов** строки             |
| `--help` | Краткая справка по команде                                 |
``` bash

sort file.txt | uniq        # удалить повторяющиеся соседние строки
sort file.txt | uniq -c     # показать количество повторов каждой строки
sort file.txt | uniq -d     # вывести только строки с повторами
sort file.txt | uniq -u     # вывести строки без повторов
sort file.txt | uniq -i     # игнорировать регистр при сравнении
uniq -s 3 file.txt          # игнорировать первые 3 символа строки
uniq -f 1 file.txt          # игнорировать первое поле (столбец)
```
