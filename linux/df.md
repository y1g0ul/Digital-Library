Показывает свободное и занятое место на файловых системах

``` bash
df [опции] [файл или точка монтирования]

если не указан файл → показывает все смонтированные файловые системы
```

|Опция|Что делает|
|---|---|
|`-h`|человеко-читаемый формат (KB, MB, GB)|
|`-H`|человеко-читаемый формат, с основанием 1000 вместо 1024|
|`-a`|показать все файловые системы, включая псевдо и пустые|
|`-T`|показать тип файловой системы|
|`-i`|показать использование inode|
|`--total`|показать общий итог по всем файловым системам|
|`-t <тип>`|показать только файловые системы указанного типа|
|`-x <тип>`|исключить файловые системы указанного типа|
Примеры:
``` bash
df -h                  # человеко-читаемый вывод по всем FS
df -H /home             # размер и свободное место для конкретного каталога
df -T                   # показать FS + тип
df -i                   # показать использование inode
df -t ext4              # показать только ext4 FS
df -x tmpfs             # исключить tmpfs из вывода
df --total -h           # общий итог по всем FS в человеко-читаемом формате
```
