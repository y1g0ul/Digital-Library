Команда в [[Linux]] для проверки условий (файлы, строки, числа)

``` bash
test условие
# или
[ условие ]
```

|Ключ|Что делает|
|---|---|
|`-e`|файл существует|
|`-f`|обычный файл|
|`-d`|каталог|
|`-r`|есть право на чтение|
|`-w`|есть право на запись|
|`-x`|есть право на выполнение|
|`-s`|файл не пустой|
|`-z`|строка пустая|
|`-n`|строка не пустая|
|`=`|строки равны|
|`!=`|строки не равны|
|`-eq`|равно (числа)|
|`-ne`|не равно (числа)|
|`-gt`|больше|
|`-lt`|меньше|
|`-ge`|больше или равно|
|`-le`|меньше или равно|
|`!`|логическое НЕ|
|`-a`|логическое И|
|`-o`|логическое ИЛИ|

``` bash
test -f file.txt              # файл существует и это обычный файл
[ -d /home ]                  # это каталог
[ -e file.txt ] && echo "есть файл"
[ -z "$VAR" ]                 # переменная пустая
[ "$A" = "$B" ]               # строки равны
[ 5 -gt 3 ]                   # 5 больше 3
[ -r file.txt ] && echo "можно читать"
[ -f file.txt -a -r file.txt ] # файл существует И доступен для чтения
```
