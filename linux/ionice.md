Команда в [[Linux]] для управления приоритетом ввода-вывода (I/O) процессов. Распределяет влияние процессов на диск. 

| Ключ | Что делает                                           |
| ---- | ---------------------------------------------------- |
| `-c` | задать класс приоритета                              |
| `-n` | задать уровень приоритета внутри класса              |
| `-p` | изменить приоритет существующего процесса            |
| `-t` | не выдавать ошибку, если приоритет нельзя установить |
| `-P` | изменить приоритет группы процессов                  |

| Класс               | Описание                                                                                                            |
| ------------------- | ------------------------------------------------------------------------------------------------------------------- |
| `0` (`none`)        | приоритет определяется планировщиком                                                                                |
| `1` (`realtime`)    | Процесс получает доступ к диску в первую очередь, независимо от остальных.                                          |
| `2` (`best-effort`) | Стандартный режим для большинства программ. Имеет 8 уровней приоритета. От 0(самого высокого) до 7(самого низкого). |
| `3` (`idle`)        | Процесс получает доступ к диску только тогда, когда диск простаивает.                                               |

``` bash
ionice -c 3 updatedb           # выполнять только при свободном диске

ionice -c 2 -n 7 rsync source dest
# низкий приоритет I/O

ionice -c 2 -n 0 tar czf backup.tar.gz /home
# высокий приоритет I/O

ionice -p 1234                 # показать приоритет процесса

ionice -c 3 -p 1234            # изменить приоритет процесса

ionice -c 2 -n 7 -p 1234       # низкий приоритет для существующего процесса
```
