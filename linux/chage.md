Команда в [[Linux]] для управления сроком действия пароля пользователя
``` bash
chage [опции] пользователь
```

|Ключ|Что делает|
|---|---|
|`-l`|показать информацию о сроках действия пароля|
|`-d`|установить дату последней смены пароля|
|`-E`|установить дату истечения учётной записи|
|`-I`|количество дней до блокировки после истечения пароля|
|`-m`|минимальное количество дней между сменами пароля|
|`-M`|максимальное количество дней действия пароля|
|`-W`|за сколько дней предупреждать об истечении пароля|
``` bash
chage -l y1g0ul               # показать параметры пароля

chage -M 90 y1g0ul            # пароль действует 90 дней

chage -m 7 y1g0ul             # минимум 7 дней между сменами

chage -W 14 y1g0ul            # предупреждать за 14 дней

chage -E 2026-12-31 y1g0ul    # срок действия аккаунта

chage -d 0 y1g0ul             # потребовать смену пароля при следующем входе

chage -I 30 y1g0ul            # блокировка через 30 дней после истечения
```
