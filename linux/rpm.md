Низкоуровневый менеджер пакетов в Red Hat [[Linux]] системах

| Команда              | Что делает                        |
| -------------------- | --------------------------------- |
| `rpm -qa`            | Список всех установленных пакетов |
| `rpm -qi pkg`        | Информация о пакете               |
| `rpm -ql pkg`        | Файлы, установленные пакетом      |
| `rpm -qf /path/file` | Какой пакет владеет файлом        |
| `rpm -ivh pkg.rpm`   | Установить пакет                  |
| `rpm -Uvh pkg.rpm`   | Обновить пакет                    |
| `rpm -e pkg`         | Удалить пакет                     |
| `rpm -V pkg`         | Проверка целостности пакета       |
| `rpm -K pkg`         | Проверка ключа пакета             |
**`rpm2cpio`** — это утилита, которая превращает `.rpm`-пакет в обычный архив `cpio`. Используется для:
- анализа пакета
- извлечения файлов
- отладки и восстановления
