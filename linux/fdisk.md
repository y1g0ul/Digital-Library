Утилита для работы с дисками и разделами в [[Linux]].
``` shell
 fdisk [options] <disk>         
```
- `-l` - показать все диски и разделы или конкретный диск

|Команда|Назначение|
|---|---|
|`m`|показать справку|
|`p`|показать таблицу разделов|
|`n`|создать новый раздел|
|`d`|удалить раздел|
|`t`|изменить тип раздела (Id)|
|`l`|список всех типов разделов|
|`w`|записать изменения и выйти|
|`q`|выйти без сохранения|
Тип раздела:
**`p`** — primary  
 Основные блоки на диске, которые могут **содержать** файловую систему. Они **напрямую видны** BIOS/OS и могут быть использованы для **загрузки** операционных систем.
- На одном диске может быть максимум 4 primary-раздела (это ограничение MBR).
- Любой primary-раздел может быть **загрузочным** (active), т.е. с него можно стартовать систему.

**`e`** — extended 
 Контейнер для логических разделов. 
- Extended сам по себе **не содержит** файловой системы, он просто держит логические разделы.
- Логические разделы внутри extended начинаются с номера **5** (например, `/dev/sda5`, `/dev/sda6` и т.д.).
- Можно создавать **сколько угодно** логических разделов внутри одного extended (по факту ограничение только размером диска и системой).
