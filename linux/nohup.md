Утилита в [[Linux]] для запуска команды, которая **игнорирует [[сигнал]] SIGHUP** (закрытие терминала).  
Позволяет процессу продолжать работу после выхода из shell или разрыва SSH-сессии.

По умолчанию вывод записывается в файл `nohup.out`.

`nohup команда [аргументы] &`

| Поведение                | Описание                                      |
| ------------------------ | --------------------------------------------- |
| Игнорирует `SIGHUP`      | Процесс не завершается при закрытии терминала |
| stdout → `nohup.out`     | Если вывод не перенаправлён                   |
| stderr → stdout          | Ошибки пишутся туда же                        |
| Работает вне job control | Не зависит от [[jobs]]/[[bg]]/[[fg]]          |
