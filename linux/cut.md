Команда в [[Linux]] которая позволяет вырезать участки на основе разделителя
- **`-d`** - по оразделителю 
- **`-f`** - по полям
Пример: `cut -d":" -f1 /etc/passwd` - вывести только имена пользователей
- **`-c`** - по символам
```
user@serv:~$ echo "abcdef" | cut -c1-3
abc
```
