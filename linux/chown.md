Команда в [[Linux]] которая меняет владельца и группу файла 
``` bash
chown [опции] владелец[:группа] файл
```

|Формат|Что означает|
|---|---|
|`user`|изменить только владельца|
|`user:group`|изменить владельца и группу|
|`:group`|изменить только группу|

|Опция|Описание|
|---|---|
|`-R`|рекурсивно изменить для каталога|
|`-v`|показать изменения|
|`-c`|показать только изменённые файлы|
|`--reference=FILE`|копировать владельца и группу с другого файла|
``` bash
sudo chown user file.txt              # изменить владельца

sudo chown user:group file.txt        # изменить владельца и группу

sudo chown :group file.txt            # изменить только группу

sudo chown -R user:group dir/         # рекурсивно для каталога

sudo chown --reference=file1 file2    # копировать владельца и группу

sudo chown -v user file.txt           # показать изменения
```
