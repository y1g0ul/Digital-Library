Утилита в [[Linux]] для создания и извлечения архивов файлов и каталогов.

``` bash
tar [ключи] archive.tar [файлы_или_каталоги]
```

**`Основные режимы`**

|Ключ|Назначение|
|---|---|
|`-c`|создать архив|
|`-t`|показать содержимое архива|
|`-x`|извлечь файлы|
|`-f`|имя файла архива (обязательный)|
**`Сжатие`**

| Ключ | Алгоритм  | Расширение |
| ---- | --------- | ---------- |
| `-z` | [[gzip]]  | `.tar.gz`  |
| `-j` | [[bzip2]] | `.tar.bz2` |
| `-J` | [[xz]]    | `.tar.xz`  |
**`Дополнительные опции`**

|Ключ|Что делает|
|---|---|
|`-v`|подробный вывод|
|`-C dir`|работать в указанном каталоге|
|`--exclude`|исключить файлы|
|`--strip-components=N`|убрать N уровней пути|
|`-p`|сохранить права|
|`--numeric-owner`|сохранить UID/GID|

Примеры:
``` bash
tar -cf foo.tar foo                # создать архив из каталога
tar -tf foo.tar                    # посмотреть содержимое архива
tar -xf foo.tar                    # извлечь архив
tar -czf foo.tar.gz foo             # создать gzip-архив
tar -xzf foo.tar.gz                 # распаковать gzip-архив
tar -xf foo.tar -C /tmp             # извлечь в каталог
tar -czf src.tar.gz src --exclude=*.o  # архив без .o файлов
tar -cf - dir | gzip > dir.tar.gz   # архив через stdout
```
