Встроенная команда оболочки ([[Linux]]) для продолжения выполнения остановленной задачи в фоновом режиме.

Используется после приостановки процесса (`Ctrl+Z`).

`bg [%номер_задачи]`

|Параметр|Описание|
|---|---|
|`%N`|Номер задания (Job ID)|
|`%+`|Текущая задача|
|`%-`|Предыдущая задача|
```bash
sleep 200            # запустить процесс
Ctrl+Z               # приостановить выполнение
bg                   # продолжить в фоне
jobs                 # посмотреть список задач
bg %1                # продолжить задачу №1 в фоне
```
