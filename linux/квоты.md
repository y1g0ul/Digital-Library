disk quotas - механизм в [[Linux]] который позволяет ограничивать использование дискового пространства пользователями и группами.

Можно ограничивать:
- объём дискового пространства
- количество файлов ([[inode]])

Квоты применяются к файловым системам и используются для контроля использования диска на серверах.

|Команда|Назначение|
|---|---|
|`quotacheck`|создать и проверить файлы квот|
|`quotaon`|включить квоты|
|`quotaoff`|отключить квоты|
|`edquota`|редактировать квоты пользователя|
|`quota`|показать квоты пользователя|
|`repquota`|отчёт по квотам|


Чтобы квоты работали, нужно включить их в /etc/[[fstab]].
`UUID=xxxxx /home ext4 defaults,usrquota,grpquota 0 2`

|Параметр|Значение|
|---|---|
|`usrquota`|квоты пользователей|
|`grpquota`|квоты групп|


**`quotacheck`**
`quotacheck [опции] файловая_система`

|Опция|Описание|
|---|---|
|`-c`|создать файлы квот|
|`-u`|проверить квоты пользователей|
|`-g`|проверить квоты групп|
|`-m`|не перемонтировать файловую систему|
**`quotaon`**
`quotaon [опции] файловая_система`

|Опция|Описание|
|---|---|
|`-a`|включить квоты на всех FS из `/etc/fstab`|
|`-u`|квоты пользователей|
|`-g`|квоты групп|
|`-v`|подробный вывод|
**`quotaoff`**
`quotaoff [опции] файловая_система`

|Опция|Описание|
|---|---|
|`-a`|отключить квоты на всех FS|
|`-u`|отключить пользовательские квоты|
|`-g`|отключить групповые квоты|
**`edquota`**
`edquota [опции] пользователь`

|Опция|Описание|
|---|---|
|`-u`|редактировать квоты пользователя|
|`-g`|редактировать квоты группы|
|`-t`|изменить grace period|
**`quota`**
`quota [опции] пользователь`

|Опция|Описание|
|---|---|
|`-u`|показать квоту пользователя|
|`-g`|показать квоту группы|
|`-v`|подробный вывод|
**`repquota`**
`repquota [опции] файловая_система`

|Опция|Описание|
|---|---|
|`-a`|отчёт по всем файловым системам|
|`-u`|отчёт по пользователям|
|`-g`|отчёт по группам|
|`-v`|подробный вывод|
При настройке квот используются два ограничения:
- `soft limit` - мягкий лимит (можно временно превышать, но не дольше чем на неделю)
- `hard limit` - жёсткий лимит (превышать нельзя)

``` bash
sudo apt install quota              # установить утилиты квот

sudo mount -o remount /home         # перемонтировать файловую систему

sudo quotacheck -cug /home          # создать файлы квот

sudo quotaon /home                  # включить квоты

sudo edquota user                   # задать лимиты пользователю

quota user                          # посмотреть квоту пользователя

repquota /home                      # отчёт по квотам

sudo quotaoff /home                 # отключить квоты
```
