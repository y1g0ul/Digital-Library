Команда в [[Linux]] для разбиения больших файлов на несколько частей меньшего размера.
``` bash 
split [опции] файл [префикс]
```

|Опция|Что делает|
|---|---|
|`-l N`|разбить на файлы по N строк|
|`-b SIZE`|разбить по SIZE байт (например `10K`, `1M`)|
|`-d`|использовать числовой суффикс вместо букв (`00`, `01`…)|
|`-a N`|длина суффикса (по умолчанию 2 буквы)|
Примеры:
``` bash
split -l 1000 large.txt    # разбить large.txt на файлы по 1000 строк
split -b 10M large.txt     # разбить на файлы по 10 мегабайт
split -d -l 500 large.txt part_  # создать part_00, part_01… по 500 строк
split -a 3 -l 200 large.txt chunk_ # суффикс из 3 букв: chunk_aaa, chunk_aab…
```

Для сброки обратно в один файл используем [[cat]]:
```
cat part_* > bigfile.txt
```
