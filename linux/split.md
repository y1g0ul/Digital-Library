Команда в [[Linux]] для разбиения больших файлов на несколько частей меньшего размера.
- Разбиение по размеру (**`-b`**  ):
`split -b 100M bigfile.tar.gz part_`  
Разделяет файл `bigfile.tar.gz` на части по 100 МБ с префиксом
- Разбиение по количеству строк (**`-l`**): 
`split -l 1000 biglog.txt log_parts_`  
Разделяет файл на части, каждая из которых содержит 1000 строк.
