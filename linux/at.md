Команда в [[Linux]] для однократного выполнения задачи в указанное время. После запуска вводятся команды и завершается ввод.
``` bash
at <время>

echo "Hello"
touch /tmp/test

Ctrl+D
```

| Команда | Что делает |
| ------- | ---------- |
| `at`    | Создать    |
| `atq`   | Посмотреть |
| `atrm`  | Удалить    |

| Ключ | Что делает                                      |
| ---- | ----------------------------------------------- |
| `-l` | показать очередь задач (аналог `atq`)           |
| `-r` | удалить задачу (аналог `atrm`)                  |
| `-f` | прочитать команды из файла                      |
| `-m` | отправить уведомление по почте после выполнения |
| `-q` | использовать указанную очередь                  |
| `-V` | показать версию                                 |
``` bash
at 15:30                     # выполнить в 15:30

at now + 10 minutes          # через 10 минут

at now + 2 hours             # через 2 часа

at tomorrow                  # завтра

at midnight                  # в полночь

at teatime                   # в 16:00

at -f script.sh now + 1 hour # выполнить скрипт через час

atq                          # показать очередь задач

at -l                        # то же самое

atrm 3                       # удалить задачу №3

at -r 3                      # то же самое
```

Правила работы аналогичны [[cron]]:
- если существует `/etc/at.allow` - доступ имеют только перечисленные пользователи
- если есть только `/etc/at.deny` - доступ имеют все, кроме перечисленных

Служба обычно называется **`atd`**.  Проверить её состояние - `systemctl status atd`.
