Команда из пакета которая показывает, какие [[процесс]]ы используют диск в реальном времени. Используется вместе с [[iostat]] для поиска процессов, которые создают высокую нагрузку на ввод-вывод.

``` bash
iotop [ключи]
```

| Ключ | Что делает                                          |
| ---- | --------------------------------------------------- |
| `-o` | показывать только процессы, которые используют диск |
| `-b` | неинтерактивный режим (для скриптов)                |
| `-n` | количество обновлений                               |
| `-d` | интервал обновления                                 |
| `-p` | отслеживать определённые PID                        |
| `-u` | показывать процессы конкретного пользователя        |
Основные поля:

| Поле         | Что показывает             |
| ------------ | -------------------------- |
| `TID`        | ID потока или [[процесс]]а |
| `PRIO`       | приоритет ввода-вывода     |
| `USER`       | пользователь процесса      |
| `DISK READ`  | скорость чтения с диска    |
| `DISK WRITE` | скорость записи на диск    |
| `COMMAND`    | команда процесса           |
``` bash
sudo iotop              # открыть мониторинг диска
sudo iotop -o           # показывать только активные процессы
sudo iotop -d 2         # обновлять каждые 2 секунды
sudo iotop -b -n 5      # вывести 5 обновлений в консоль
sudo iotop -p 1234      # отслеживать процесс с PID 1234
```
