Утилита в [[Linux]] для изменения параметров файловых систем **`ext2/ext3/ext4`**

Позволяет:
- изменять параметры файловой системы
- настраивать интервалы проверки [[fsck]]
- изменять количество зарезервированных блоков
- добавлять или отключать функции файловой системы

| Опция | Описание                                                |
| ----- | ------------------------------------------------------- |
| `-l`  | показать параметры файловой системы                     |
| `-c`  | установить количество монтирований до проверки [[fsck]] |
| `-i`  | установить интервал времени между проверками            |
| `-m`  | задать процент зарезервированных блоков                 |
| `-L`  | установить метку файловой системы (label)               |
| `-U`  | изменить UUID файловой системы                          |
| `-o`  | изменить параметры монтирования по умолчанию            |
| `-e`  | задать поведение при ошибке файловой системы            |
``` bash
sudo tune2fs -l /dev/sdb1                 # показать параметры файловой системы

sudo tune2fs -L DATA /dev/sdb1            # задать label файловой системы

sudo tune2fs -m 1 /dev/sdb1               # изменить резервные блоки до 1%

sudo tune2fs -c 20 /dev/sdb1              # проверка fsck каждые 20 монтирований

sudo tune2fs -i 30d /dev/sdb1             # проверка fsck каждые 30 дней

sudo tune2fs -U random /dev/sdb1          # сгенерировать новый UUID
```
