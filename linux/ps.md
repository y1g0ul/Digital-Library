Команда в [[Linux]] для просмотра [[процесс]]ов

```bash
ps [опции]
```

| Опция           | Описание                               |
| --------------- | -------------------------------------- |
| `a`             | Процессы всех пользователей (с TTY)    |
| `u`             | Детальный вывод (USER, CPU, MEM и др.) |
| `x`             | Процессы без терминала (демоны)        |
| `-e`            | Показать все процессы                  |
| `-f`            | Полный формат вывода                   |
| `-o <поля>`     | Задать формат вывода                   |
| `-p <PID>`      | Показать конкретный процесс            |
| `--sort <поле>` | Сортировка вывода                      |
| `--forest`      | Показать дерево процессов              |
Примеры:
``` bash
ps                      # процессы текущего терминала
ps aux                  # все процессы (классическая форма)
ps -ef                  # все процессы в полном формате
ps -p 1234              # информация о процессе с PID 1234
ps -o pid,user,cmd      # пользовательский формат вывода
ps aux --sort=-%cpu     # сортировка по CPU (убывание)
ps -ef --forest         # дерево процессов
```
