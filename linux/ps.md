Команда в [[Linux]] для просмотра [[процесс]]ов

```bash
y1g0ul@LAPTOP-MSSNKT4E:~$ ps -efl
F S UID        PID  PPID  C PRI  NI ADDR SZ WCHAN  STIME TTY          TIME CMD
0 S root         1     0  0  80   0 -  2238 ?      10:21 ?        00:00:00 /init
0 S root         8     1  0  80   0 -  2238 ?      10:21 tty1     00:00:00 /init
0 S y1g0ul       9     8  0  80   0 -  3528 ?      10:21 tty1     00:00:00 -bash
0 R y1g0ul      58     9  0  80   0 -  4090 ?      10:57 tty1     00:00:00 ps -efl
```

`-e`
- Показывает все [[процесс]]ы в системе.
- Эквивалентно `--everyone` или `--all`.
`-f`
- Полный формат вывода (full-format).    
- Включает информацию о родительском [[процесс]]е ([[PPID]]), пользователе ([[uid]]), времени запуска, команде и др.    
`-l`
- Длинный формат (long format).
- Показывает дополнительные поля: приоритет (PRI), nice value (NI), состояние [[процесс]]а (S), идентификатор группы (PGID), идентификатор сессии (SID), используемую память и т.д.
