Команда в [[Linux]] для трассировки системных вызовов и сигналов [[процесс]]ов.
``` bash
strace [опции] команда

strace [опции] -p PID
```

|Ключ|Что делает|
|---|---|
|`-p`|подключиться к уже запущенному процессу|
|`-o`|записать вывод в файл|
|`-f`|отслеживать дочерние процессы|
|`-e`|фильтровать системные вызовы|
|`-c`|показать статистику системных вызовов|
|`-t`|добавить время вызова|
|`-tt`|добавить время с микросекундами|
|`-T`|показать время выполнения системного вызова|
|`-s`|ограничить длину выводимых строк|
|`-v`|подробный вывод|
|`-y`|показывать пути файловых дескрипторов|
![[strace_work]]

``` bash
strace ls
strace cat file.txt
strace -p 1234
strace -f ./script.sh
strace -o trace.log ls
strace -c ls # Статистика вызовов
strace -e openat ls # Показать только обращения к файлам
strace -e read,write cat file.txt
```

**`Часто встречающиеся системные вызовы`**

|Вызов|Описание|
|---|---|
|`open()` / `openat()`|открытие файла|
|`read()`|чтение данных|
|`write()`|запись данных|
|`close()`|закрытие файла|
|`execve()`|запуск программы|
|`fork()`|создание процесса|
|`clone()`|создание потока или процесса|
|`mmap()`|отображение памяти|
|`stat()`|получение информации о файле|
|`socket()`|создание сокета|
|`connect()`|подключение к сокету|
