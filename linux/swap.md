Это место на диске, которое [[Linux]] использует как расширение оперативной памяти.

**`mkswap`** — подготовка swap.
Размечает область как swap:
- записывает сигнатуру
- создаёт swap-заголовок
- задаёт UUID
Аналог [[mkfs]], но для swap.

**`swapon`** — включение swap.
- регистрирует swap в ядре
- ядро начинает использовать его при необходимости
`-a` - включить всё из /etc/[[fstab]]
`-s` - проверка активных swap
`--show` - посмотреть приоритет

**`swapoff`** — выключение swap
- возвращает данные из swap обратно в RAM
- затем отключает swap
