Механизм в [[Linux]], который определяет язык интерфейса, формат даты и времени, кодировку, денежные единицы, сортировку текста и другие региональные настройки.

| Термин   | Что означает                                                                                 |
| -------- | -------------------------------------------------------------------------------------------- |
| `i18n`   | Интернационализация (internationalization) — подготовка программы к работе с разными языками |
| `l10n`   | Локализация (localization) — перевод программы и настройка под конкретный регион             |
| `locale` | Набор региональных настроек системы(`язык_страна.КОДИРОВКА`)                                 |


**`Переменные окружения locale`**
- `LANG` основная локаль системы
- `LANGUAGE` предпочтительный язык сообщений
- `LC_ALL` переопределяет все настройки locale
- `LC_TIME` формат даты и времени
- `LC_NUMERIC` формат чисел
- `LC_MONETARY` денежные единицы
- `LC_MESSAGES` язык сообщений программ
- `LC_COLLATE` сортировка текста
- `LC_CTYPE` работа с символами и кодировками
Приоритет:
``` bash
LC_ALL
↓
LC_*
↓
LANG
```


**`locale`**
Команда для просмотра и изменения локали текущей сессии.

| Ключ | Что делает                    |
| ---- | ----------------------------- |
| `-a` | показать доступные локали     |
| `-m` | показать доступные кодировки  |
| `-k` | показать подробную информацию |


**`localectl`**
Команда [[systemd]] для управления локализацией
``` bash
localectl [команда]
```

| Команда        | Что делает                 |
| -------------- | -------------------------- |
| `status`       | текущие настройки          |
| `list-locales` | доступные локали           |
| `set-locale`   | изменить локаль            |
| `set-keymap`   | изменить раскладку консоли |

**`Генерация локалей`**
Cоздание файлов с региональными настройками, которые программы потом используют для работы.
`/etc/locale.gen` - файл с локалями. Система читает его и создает необходимые данные. 

Debian/Ubuntu:
``` bash
sudo dpkg-reconfigure locales
sudo locale-gen ru_RU.UTF-8
```
RHEL/CentOS/Fedora:
``` bash
sudo localectl set-locale LANG=en_US.UTF-8
```
