Команда в [[Linux]] для создания директории.

``` bash
mkdir [опции] каталог
```

| Опция       | Описание                                        |
| ----------- | ----------------------------------------------- |
| `-p`        | Создать родительские каталоги при необходимости |
| `-v`        | Показать сообщение о создании                   |
| `-m MODE`   | Установить права доступа                        |
| `--help`    | Показать помощь                                 |
| `--version` | Версия команды                                  |
``` bash
mkdir test                 # создать каталог
mkdir dir1 dir2            # создать несколько каталогов
mkdir -p project/src       # создать вложенные каталоги
mkdir -v logs              # показать сообщение о создании
mkdir -m 755 public        # создать каталог с правами 755
```
