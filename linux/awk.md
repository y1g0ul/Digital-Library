Команда в [[Linux]] для обработки текста по шаблонам и полям

``` bash
awk 'шаблон {действие}' файл
```

| Ключ         | Что делает                  |
| ------------ | --------------------------- |
| `-F`         | задать разделитель полей    |
| `-v`         | передать переменную         |
| `-f`         | читать программу из файла   |
| `$1, $2 ...` | обращение к полям           |
| `$0`         | вся строка                  |
| `NF`         | количество полей            |
| `NR`         | номер строки                |
| `BEGIN`      | выполняется до обработки    |
| `END`        | выполняется после обработки |
| `print`      | вывести данные              |
| `printf`     | форматированный вывод       |
``` bash
awk '{print $1}' file.txt           # вывести первый столбец

awk -F: '{print $1}' /etc/passwd    # разделитель :

awk '{print NF}' file.txt           # количество полей в строке

awk '{print NR, $0}' file.txt       # номер строки + строка

awk 'BEGIN {print "Start"} {print $1} END {print "End"}' file.txt

awk '$3 > 100 {print $1}' file.txt  # условие

awk -v name="Nikita" '{print name, $1}' file.txt

awk '{sum += $1} END {print sum}' numbers.txt  # сумма`
```
