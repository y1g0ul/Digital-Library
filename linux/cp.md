Команда в [[Linux]] для копирования. 
``` bash
cp [опции] источник назначение
```

| Ключ        | Что делает                                                |
| ----------- | --------------------------------------------------------- |
| `-r`, `-R`  | копировать каталоги рекурсивно                            |
| `-a`        | архивный режим (сохраняет права, владельца, время и т.д.) |
| `-i`        | спрашивать перед перезаписью                              |
| `-f`        | принудительная перезапись                                 |
| `-n`        | не перезаписывать существующие файлы                      |
| `-u`        | копировать только если файл новее                         |
| `-v`        | подробный вывод                                           |
| `-p`        | сохранить права и время                                   |
| `--parents` | сохранять структуру каталогов                             |
| `-t`        | указать каталог назначения                                |
``` bash
cp file.txt copy.txt             # копировать файл

cp file.txt /tmp/                # копировать в каталог

cp -r dir1 dir2                  # копировать каталог

cp -a dir1 backup/               # архивное копирование

cp -i file.txt copy.txt          # спросить перед перезаписью

cp -u file.txt /backup/          # только если новее

cp -v file.txt /tmp/             # показать процесс

cp --parents dir1/file.txt /tmp/ # сохранить структуру пути
```
