Команда в [[Linux]] для создания и управления терминальными сессиями (terminal multiplexer). Современный аналог [[screen]]. 
``` bash
tmux [команда]
```

| Ключ                | Что делает                    |
| ------------------- | ----------------------------- |
| `new -s`            | создать новую сессию          |
| `ls`                | показать список сессий        |
| `attach -t`         | подключиться к сессии         |
| `detach`            | отключиться от сессии         |
| `kill-session -t`   | удалить сессию                |
| `rename-session -t` | переименовать сессию          |
| `split-window -h`   | разделить окно по горизонтали |
| `split-window -v`   | разделить окно по вертикали   |
| `new-window`        | создать новое окно            |
| `kill-window`       | удалить окно                  |


Основные сочетания клавиш:

|Сочетание|Что делает|
|---|---|
|`Ctrl+b d`|отключиться от сессии|
|`Ctrl+b c`|создать новое окно|
|`Ctrl+b n`|следующее окно|
|`Ctrl+b p`|предыдущее окно|
|`Ctrl+b %`|разделить окно по вертикали|
|`Ctrl+b "`|разделить окно по горизонтали|
|`Ctrl+b o`|переключиться между панелями|
|`Ctrl+b x`|закрыть панель|
|`Ctrl+b [`|режим прокрутки|
|`Ctrl+b ?`|справка по сочетаниям|
``` bash
tmux                           # создать новую сессию

tmux new -s work               # создать сессию work

tmux ls                        # показать все сессии

tmux attach -t work            # подключиться к сессии

tmux kill-session -t work      # удалить сессию

tmux new-window                # создать окно

tmux split-window -h           # разделить окно по горизонтали

tmux split-window -v           # разделить окно по вертикали
```
