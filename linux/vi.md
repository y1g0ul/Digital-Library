Классический текстовый редактор в [[Linux]], работающий в терминале.  
Поддерживает **модальную работу** (разные режимы ввода и управления).

Часто в современных системах используется его расширенная версия - `vim`.

``` bash
vi [файл]
```
Если файл существует - откроется для редактирования.  
Если нет - будет создан при сохранении.

|Режим|Назначение|
|---|---|
|Normal|Навигация и команды|
|Insert|Ввод текста|
|Command (`:`)|Сохранение, выход и др.|
**`Перемещение по тексту (Normal)`**

| Команда | Действие                   |
| ------- | -------------------------- |
| `h`     | Влево                      |
| `j`     | Вниз                       |
| `k`     | Вверх                      |
| `l`     | Вправо                     |
| `w`     | В начало следующего слова  |
| `b`     | В начало предыдущего слова |
| `e`     | В конец слова              |
| `(`     | В начало предложения       |
| `)`     | В конец предложения        |
| `{`     | В начало абзаца            |
| `}`     | В конец абзаца             |
| `^`     | В начало строки            |
| `$`     | В конец строки             |
| `gg`    | В начало файла             |
| `G`     | В конец файла              |
| `1G`    | Перейти к строке 1         |
| `:N`    | Перейти к строке N         |
**`Ввод текста`**

|Команда|Действие|
|---|---|
|`i`|Вставка перед курсором|
|`a`|Вставка после курсора|
|`o`|Новая строка ниже|
|`O`|Новая строка выше|
|`Esc`|Выход в Normal|
**`Копирование (yank)`**

|Команда|Действие|
|---|---|
|`yy`|Скопировать строку|
|`yw`|Скопировать слово|
|`yl`|Скопировать символ|
|`y$`|Скопировать до конца строки|
**`Удаление (вырезание)`**

|Команда|Действие|
|---|---|
|`dd`|Удалить строку|
|`dw`|Удалить слово|
|`dl`|Удалить символ|
|`d$`|Удалить до конца строки|
|`x`|Удалить символ под курсором|
**`Вставка`**

|Команда|Действие|
|---|---|
|`p`|Вставить после курсора|
|`P`|Вставить перед курсором|
**`Поиск`**

|Команда|Действие|
|---|---|
|`/текст`|Поиск вперёд|
|`?текст`|Поиск назад|
|`n`|Следующее совпадение|
|`N`|Предыдущее совпадение|
**`Отмена и повтор`**

|Команда|Действие|
|---|---|
|`u`|Отменить|
|`Ctrl + r`|Повторить|
|`.`|Повторить последнюю команду|
**`Сохранение и выход`**

| Команда       | Действие                         |
| ------------- | -------------------------------- |
| `:w`          | Сохранить                        |
| `:q`          | Выйти                            |
| `:wq`         | Сохранить и выйти                |
| `:q!`         | Выйти без сохранения             |
| `:w filename` | Сохранить как                    |
| `ZZ`          | Сохранить (если изменён) и выйти |
| `ZQ`          | Выйти без сохранения             |
``` bash
vi file.txt        # открыть файл
i                  # перейти в режим вставки
Esc                # выйти из режима вставки
:w                 # сохранить
:q                 # выйти
:wq                # сохранить и выйти
:q!                # выйти без сохранения
```
