Классический текстовый редактор в [[Linux]], работающий в терминале.  
Поддерживает **модальную работу** (разные режимы ввода и управления).

Часто в современных системах используется его расширенная версия - `vim`.

``` bash
vi [файл]
```
Если файл существует - откроется для редактирования.  
Если нет - будет создан при сохранении.

|Режим|Назначение|
|---|---|
|Normal|Управление и навигация|
|Insert|Ввод текста|
|Command|Команды сохранения, выхода и т.д.|

|Команда|Действие|
|---|---|
|`h`|Влево|
|`j`|Вниз|
|`k`|Вверх|
|`l`|Вправо|
|`b`|В начало слова|
|`e`|В конец слова|
|`(`|В начало предложения|
|`)`|В конец предложения|
|`{`|В начало абзаца|
|`}`|В конец абзаца|
|`^`|В начало строки|
|`$`|В конец строки|
|`1G`|В начало файла|
|`G`|В конец файла|

| Команда  | Описание                             |
| -------- | ------------------------------------ |
| `i`      | Режим вставки до текущего символа    |
| `a`      | Режим вставки после текущего символа |
| `Esc`    | Вернуться в Normal режим             |
| `dd`     | Удалить строку                       |
| `yy`     | Копировать строку                    |
| `p`      | Вставить                             |
| `u`      | Отменить                             |
| `/текст` | Поиск                                |
| `:w`     | Сохранить                            |
| `:q`     | Выйти                                |
| `:wq`    | Сохранить и выйти                    |
| `:q!`    | Выйти без сохранения                 |


``` bash
vi file.txt        # открыть файл
i                  # перейти в режим вставки
Esc                # выйти из режима вставки
:w                 # сохранить
:q                 # выйти
:wq                # сохранить и выйти
:q!                # выйти без сохранения
```
