| Ключ       | Что делает                                                      |
| ---------- | --------------------------------------------------------------- |
| `export`   | делает переменную доступной для дочерних процессов ([[export]]) |
| `unset`    | удаляет переменную ([[unset]])                                  |
| `env`      | показывает переменные окружения ([[env]])                       |
| `printenv` | показать значение переменной                                    |
| `set`      | показать все переменные (включая локальные)                     |
| `$VAR`     | получить значение переменной                                    |
| `$(...)`   | подставить результат команды                                    |
| `readonly` | сделать переменную неизменяемой                                 |
``` bash
VAR=hello                    # создать переменную
echo $VAR                    # вывести значение

export VAR=hello             # сделать переменную окружения
env | grep VAR               # проверить

unset VAR                    # удалить переменную

NAME="Nikita"                # строка
echo "Hello $NAME"           # подстановка

DATE=$(date)                 # результат команды в переменную
echo $DATE

readonly PI=3.14             # неизменяемая переменная
```
