Утилита в [[Linux]] для **динамического мониторинга процессов в реальном времени**.  
Показывает загрузку CPU, использование памяти, uptime, количество процессов и их состояние.

`top [опции]`


Основные опции

| Опция      | Описание                       |
| ---------- | ------------------------------ |
| `-d N`     | Интервал обновления (секунды)  |
| `-u user`  | Показать процессы пользователя |
| `-p PID`   | Отслеживать конкретный процесс |
| `-n N`     | Выполнить N обновлений и выйти |
| `-b`       | Batch-режим (для скриптов)     |
| `-o FIELD` | Сортировка по полю             |
Основные интерактивные клавиши

|Клавиша|Действие|
|---|---|
|`q`|Выход|
|`k`|Завершить процесс|
|`r`|Изменить приоритет (renice)|
|`P`|Сортировка по CPU|
|`M`|Сортировка по памяти|
|`T`|Сортировка по времени работы|
|`h`|Помощь|
``` bash
top                       # запустить мониторинг
top -d 1                  # обновление каждую секунду
top -u user               # процессы пользователя
top -p 1234               # отслеживать PID 1234
top -n 1                  # один вывод и выход
top -b -n 1 > report.txt  # сохранить вывод в файл
```
