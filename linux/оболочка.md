В [[Linux]] используются разные оболочки

|Оболочка|Путь|Комментарий|
|---|---|---|
|sh|`/bin/sh`|Классический Bourne Shell (минимальный)|
|bash|`/bin/bash`|Самая популярная (по умолчанию в Ubuntu)|
|ksh|`/bin/ksh`|Korn Shell, мощный, но редкий|
|csh|`/bin/csh`|C-подобный синтаксис|
|tcsh|`/bin/tcsh`|Улучшенный csh|

Увидеть какую оболочку использует пользователь мы можем в файле /etc/[[passwd]]

**`Глобальные настройки (для всех пользователей)`**
- `/etc/profile` - Общие настройки для login shell
- `/etc/bash.bashrc` - Общие настройки bash

**`Пользовательские настройки`**
- `~/.profile` - Login shell (универсальный)
- `~/.bash_profile` - Login shell (bash)
- `~/.bashrc` - Interactive shell
