В [[Linux]] используются разные оболочки

The Bourne shell - `/bin/sh`
The Bourne again shell - `/bin/bash`
The Korn shell - `/bin/ksh`
The C shell - `/bin/csh`
Tom's C shell - `/bin/tcsh`

Увидеть какую оболочку использует пользователь мы можем в файле /etc/[[passwd]]

Глобальные настройки оболочки находятся в файле `/etc/profile`.
