Команда для того что бы выполнить команду в запущенном [[container]] в [[docker]].
```
docker exec [OPTIONS] CONTAINER COMMAND [ARG...]
```
- `-i` - Оставляет stdin открытым, позволяет вводить команды с клавиатуры
- `-t` - Выделяет виртуальные терминал, делает нормальный интерактивный интерфейс
