Команда для просмотра запущенных [[container]] в [[docker]].
``` shell
docker ps [OPTIONS]
```
- `-a` - показать все контейнеры вне зависимости от того запущены они или нет
- `-q` - показать только идентификаторы [[container]]
- `-f` - показать [[container]] по фильтру

| Фильтр         | Пример                            | Что делает                                                                                                                                          |
| -------------- | --------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| `status=`      | `docker ps -f status=running`     | Показать только контейнеры с указанным статусом (`created`, `restarting`, `running`, `paused`, `exited`)                                            |
| `name=`        | `docker ps -f name=web`           | Показать контейнеры, имя которых содержит `web`                                                                                                     |
| `id=`          | `docker ps -f id=<container_id>`  | Показать контейнер по ID                                                                                                                            |
| `ancestor=`    | `docker ps -f ancestor=nginx`     | Показать контейнеры, созданные из образа `nginx`                                                                                                    |
| `label=`       | `docker ps -f label=env=prod`     | Показать контейнеры с определённой меткой (label)                                                                                                   |
| `before=`      | `docker ps -f before=<container>` | Показать контейнеры, созданные **до указанного** контейнера                                                                                         |
| `since=`       | `docker ps -f since=<container>`  | Показать контейнеры, созданные **после указанного** контейнера                                                                                      |
| `exited=<код>` | `docker ps -a -f exited=130`      | Показывает контейнеры, которые **завершились с указанным кодом выхода**. Используется для поиска контейнеров, завершившихся с ошибкой или аварийно. |

| Код `exited` | Значение                            |
| ------------ | ----------------------------------- |
| 0            | Успешно                             |
| 1            | Ошибка                              |
| 130          | Прервано сигналом `SIGINT` (Ctrl+C) |
| 137          | Завершено сигналом `SIGKILL`        |
