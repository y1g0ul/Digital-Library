позволяет [[container]] находить друг друга по имени, безопасно обмениваться данными и изолировать сервисы друг от друга

Основные типы сетей:
**`bridge (по умолчанию)`**
- Создаётся автоматически
- Контейнеры могут выходить в интернет

**`user-defined bridge`**
- Есть встроенный DNS
- Контейнеры находят друг друга по имени

**`host`**
- Контейнер использует сеть хоста напрямую
- Нет изоляции
- Порты не пробрасываются

**`none`**
- Сеть полностью отключена
- Нет доступа ни к чему

|Команда|Что делает|
|---|---|
|`docker network ls`|список сетей|
|`docker network inspect mynet`|детали сети|
|`docker network create mynet`|создать сеть|
|`docker network rm mynet`|удалить сеть|
|`docker network connect mynet c1`|подключить контейнер|
|`docker network disconnect mynet c1`|отключить контейнер|
