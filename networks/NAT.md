Network Address Translation - механизм преобразования IP-адресов. Чаще всего NAT используется для того, чтобы устройства с частными (серыми) [[networks/IP]]-адресами могли выходить в интернет через один публичный (белый) [[networks/IP]]-адрес.
``` bash
# Пример запроса к сайту
Ноутбук
192.168.1.100:54321
        ↓
Роутер
89.208.15.42:40001
        ↓
Google
142.250.x.x:443

# Роутер запоминает соответствие
192.168.1.100:54321 - 89.208.15.42:40001

# Когда приходит ответ на 
89.208.15.42:40001

# Роутер понимает нужно вернуть пакет на 
192.168.1.100:54321
```

Внутри роутера хранится таблица преобразований.

|Внутренний адрес|Внешний адрес|
|---|---|
|192.168.1.100:54321|89.208.15.42:40001|
|192.168.1.101:54322|89.208.15.42:40002|
